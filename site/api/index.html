API эндпоинты

Базовый путь: /api/

CRUD-эндпоинты

Стандартные действия DRF: - GET / — список - POST / — создание - GET /{id}/ — детальная запись - PUT/PATCH /{id}/ — обновление - DELETE /{id}/ — удаление

Ресурсы: - /api/teachers/ — учителя - /api/students/ — ученики - /api/schoolclasses/ — классы - /api/subjects/ — предметы - /api/classrooms/ — кабинеты - /api/grades/ — оценки - /api/schedule/ — расписание - /api/assignments/ — назначения учителей

Отчеты и аналитика

Предмет в расписании

GET /api/reports/subject_for_class/?class_id=1&day=1&lesson=2

Сколько учителей на каждый предмет

GET /api/reports/teachers_per_subject/

Учителя, ведущие те же предметы, что и учитель информатики

GET /api/reports/same_subject_teachers/?class_id=1

Количество мальчиков и девочек в каждом классе

GET /api/reports/gender_count_per_class/

Статистика кабинетов

GET /api/reports/classroom_stats/

Отчет об успеваемости класса

GET /api/reports/class_performance_report/?class_id=1&quarter=1&school_year=2023-2024

Аутентификация и пользователи (Djoser)

Базовый путь: /api/auth/

Ключевые эндпоинты: - POST /api/auth/users/ — регистрация - POST /api/auth/token/login/ — получение токена - POST /api/auth/token/logout/ — отзыв токена - GET /api/auth/users/me/ — текущий пользователь

Токен в заголовке:

Authorization: Token <your_token>
