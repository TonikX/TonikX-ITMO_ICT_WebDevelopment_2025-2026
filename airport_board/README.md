# Табло отображения информации об авиаперелетах

Веб-приложение для управления информацией о рейсах аэропорта, бронирования мест и оставления отзывов.

## Описание проекта

Система позволяет:
- Просматривать информацию о рейсах (номер рейса, авиакомпания, время отлета/прилета, тип, номер гейта)
- Регистрировать новых пользователей
- Просматривать и резервировать места на рейсах
- Редактировать и удалять свои резервирования
- Регистрировать пассажиров на рейсы через Django Admin (для администратора)
- Просматривать список всех пассажиров рейса в табличном виде
- Писать отзывы о рейсах с указанием даты рейса, текста комментария и рейтинга (1-10)

## Требования

- Python 3.8+
- Django 5.2.7

## Установка и запуск

### 1. Убедитесь, что у вас установлен Python 3.8+

```bash
python --version
```

### 2. Установите Django (если еще не установлен)

```bash
pip install django
```

### 3. Перейдите в директорию проекта

```bash
cd airport_board
```

### 4. База данных уже создана и мигрирована

Миграции уже применены. Если нужно сбросить БД:

```bash
python manage.py migrate
```

### 5. Загрузите тестовые данные (опционально)

```bash
python manage.py populate_data
```

Эта команда создаст 5 тестовых рейсов.

### 6. Создайте администратора

```bash
python manage.py createsuperuser
```

Введите:
- Имя пользователя: `admin`
- Email: `admin@example.com`
- Пароль: (придумайте свой)

### 7. Запустите сервер

```bash
python manage.py runserver
```
### 8. Откройте браузер

- **Главная страница**: http://127.0.0.1:8000/
- **Панель администратора**: http://127.0.0.1:8000/admin/

## Структура проекта

### Модели данных

#### Flight (Рейс)
- `flight_number` - номер рейса (уникальный)
- `airline` - авиакомпания
- `departure_time` - время отлета
- `arrival_time` - время прилета
- `flight_type` - тип рейса (прилет/отлет)
- `gate_number` - номер гейта
- `total_seats` - общее количество мест

#### Reservation (Резервирование)
- `user` - пользователь, создавший резервирование
- `flight` - рейс
- `seat_number` - номер места
- `created_at` - дата и время создания резервирования
- `is_active` - флаг активности резервирования

#### Passenger (Пассажир)
- `reservation` - связь с резервированием
- `first_name` - имя
- `last_name` - фамилия
- `ticket_number` - номер билета (уникальный)
- `passport_number` - номер паспорта

#### Review (Отзыв)
- `flight` - рейс
- `user` - автор отзыва
- `flight_date` - дата рейса
- `comment` - текст комментария
- `rating` - рейтинг (1-10)
- `created_at` - дата создания отзыва

## Функционал

### Для обычных пользователей:

1. **Регистрация** (`/register/`)
   - Создание нового аккаунта с указанием имени пользователя, email, имени и фамилии

2. **Просмотр рейсов** (`/`)
   - Список всех рейсов с основной информацией
   - Отображение среднего рейтинга для каждого рейса
   - Информация о доступных местах

3. **Детальная информация о рейсе** (`/flight/<id>/`)
   - Полная информация о рейсе
   - Список отзывов
   - Кнопки для бронирования и добавления отзыва

4. **Бронирование места** (`/flight/<id>/reserve/`)
   - Выбор номера места
   - Проверка доступности мест
   - Защита от двойного бронирования

5. **Управление бронированиями** (`/reservations/`)
   - Список всех активных бронирований пользователя
   - Редактирование номера места
   - Отмена бронирования

6. **Просмотр пассажиров** (`/flight/<id>/passengers/`)
   - Таблица со всеми зарегистрированными пассажирами рейса
   - Информация о номерах билетов и местах

7. **Добавление отзывов** (`/flight/<id>/review/`)
   - Написание отзыва о рейсе
   - Указание даты рейса и рейтинга (1-10)

### Для администраторов:

Доступ к Django Admin (`/admin/`):

1. **Управление рейсами**
   - Создание, редактирование и удаление рейсов
   - Просмотр статистики по местам

2. **Управление резервированиями**
   - Просмотр всех резервирований
   - Фильтрация по рейсам и пользователям

3. **Регистрация пассажиров**
   - Добавление информации о пассажире к резервированию
   - Ввод номера билета и паспортных данных

4. **Управление отзывами**
   - Модерация отзывов
   - Просмотр статистики по рейтингам

## URL-маршруты

- `/` - главная страница со списком рейсов
- `/register/` - регистрация нового пользователя
- `/accounts/login/` - вход в систему
- `/accounts/logout/` - выход из системы
- `/flight/<id>/` - детальная информация о рейсе
- `/flight/<id>/reserve/` - бронирование места на рейсе
- `/flight/<id>/passengers/` - список пассажиров рейса
- `/flight/<id>/review/` - добавление отзыва к рейсу
- `/reservations/` - список бронирований пользователя
- `/reservation/<id>/edit/` - редактирование бронирования
- `/reservation/<id>/delete/` - удаление бронирования
- `/admin/` - панель администратора
