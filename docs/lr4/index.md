# Документация клиентской части

## Обзор
Клиентская часть системы управления гостиницей разработана на Vue.js 3 с использованием Vuetify.

## Основные возможности
- ✅ Авторизация и регистрация пользователей
- ✅ Управление гостями отеля
- ✅ Просмотр и бронирование номеров
- ✅ Личный кабинет пользователя


## Быстрый старт
```bash
# Клонирование проекта
cd hotel_frontend

# Установка зависимостей
npm install

# Запуск в режиме разработки
npm run dev

# Сборка для продакшена
npm run build
```
