\## 🎯 Краткое описание



В рамках лабораторной работы была разработана \*\*"Доска домашних заданий"\*\* - веб-приложение для управления учебным процессом. Система позволяет преподавателям публиковать задания, а студентам - сдавать решения и отслеживать оценки.



\### 🚀 Ключевые возможности



\- ✅ \*\*Регистрация и аутентификация\*\* пользователей

\- ✅ \*\*Публикация и просмотр\*\* домашних заданий

\- ✅ \*\*Сдача решений\*\* в текстовом формате

\- ✅ \*\*Система оценок\*\* с комментариями преподавателя

\- ✅ \*\*Таблица успеваемости\*\* всех студентов

\- ✅ \*\*Поиск и фильтрация\*\* заданий

\- ✅ \*\*Пагинация\*\* страниц

\- ✅ \*\*Адаптивный дизайн\*\* на Bootstrap 5



\### 🛠 Технологический стек



\- \*\*Backend:\*\* Django 3.2

\- \*\*Frontend:\*\* HTML5, CSS3, Bootstrap 5

\- \*\*Database:\*\* SQLite3

\- \*\*Дополнительно:\*\* Django Crispy Forms, Font Awesome
