# Лабораторная работа 1: Работа с сокетами

## Обзор

Данный проект представляет собой практическую реализацию пяти заданий по работе с сокетами в Python. Все задания выполнены с использованием стандартной библиотеки Python.

### Цели работы

- Изучение основ сетевого программирования
- Практическое применение библиотеки socket
- Реализация клиент-серверной архитектуры
- Работа с различными протоколами (UDP, TCP, HTTP)
- Освоение многопоточного программирования

### Технические требования

- Python 3.8+
- Библиотеки: socket, threading, json, math, urllib.parse
- Протоколы: UDP, TCP, HTTP

## Запуск программ

### Главное меню

```bash

```

### Отдельные задания

