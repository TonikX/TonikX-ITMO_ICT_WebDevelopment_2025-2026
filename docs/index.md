## 🎯 О проекте

Разработана полнофункциональная система управления домашними заданиями с использованием Django 3.2.

## 📋 Реализованный функционал

### 👨‍🎓 Для студентов:
- ✅ Регистрация и вход в систему
- ✅ Просмотр всех заданий
- ✅ Поиск и фильтрация заданий
- ✅ Сдача решений в текстовом виде
- ✅ Просмотр своих оценок

### 👨‍🏫 Для преподавателей:
- ✅ Создание и редактирование заданий
- ✅ Оценка работ студентов
- ✅ Комментирование работ
- ✅ Таблица успеваемости всех студентов
- ✅ Админ-панель для управления

## 🛠 Технологический стек

| Компонент | Технологии |
|-----------|------------|
| **Backend** | Django 3.2, Python |
| **Frontend** | HTML5, CSS3, Bootstrap 5 |
| **Database** | SQLite3 |
| **Дополнительно** | Font Awesome, Django Crispy Forms |

## ⭐ Особенности реализации

- **Адаптивное меню** на Bootstrap 5
- **Пагинация** страниц с заданиями
- **Поиск и фильтрация**, совместимые с пагинацией
- **Разделение прав** доступа
- **Цветовое кодирование** оценок

## Архитектура и структура проекта

```bash
laboratory_work_2/
│
├── homework_board/                 # Настройки Django проекта
│   ├── __init__.py
│   ├── settings.py                # Основные настройки Django
│   ├── urls.py                    # Главные URL-маршруты
│   ├── wsgi.py                    # Конфигурация WSGI
│   └── asgi.py                    # Конфигурация ASGI
│
├── assignments/                   # Основное приложение
│   ├── __init__.py
│   ├── admin.py                   # Регистрация моделей в админ-панели
│   ├── apps.py                    # Конфигурация приложения
│   ├── forms.py                   # Формы регистрации и сдачи заданий
│   ├── models.py                  # Модели данных (Subject, Assignment, Submission)
│   ├── views.py                   # Представления (логика обработки запросов)
│   ├── urls.py                    # URL-маршруты приложения
│   │
│   ├── migrations/                # Миграции базы данных
│   │   ├── __init__.py
│   │   └── 0001_initial.py       # Первоначальные миграции
│   │
│   └── templatetags/              # Кастомные теги шаблонов
│       ├── __init__.py

│
├── templates/                     # HTML-шаблоны (клиентская часть)
│   ├── base.html                 # Базовый шаблон (меню, шапка, подключение стилей)
│   │
│   ├── assignments/              # Страницы, связанные с заданиями

│       ├── login.html            # Форма входа пользователя

│
├── static/                       # Статические файлы
│   └── css/
│       └── admin_custom.css      # Кастомные стили для админ-панели
│
├── .github/                      # GitHub Actions workflow
│   └── workflows/
│       └── ci.yml               # Автоматический деплой документации
│
├── docs/                         # Документация проекта
│   └── index.md                 # Отчет по лабораторной работе
│
├── db.sqlite3                    # Локальная база данных
├── manage.py                     # Точка входа в Django-приложение
└── requirements.txt              # Зависимости проекта

```

## 🚀 Запуск проекта

```bash
# Установка зависимостей
pip install django==3.2.* django-crispy-forms crispy-bootstrap5

# Применение миграций
python manage.py makemigrations
python manage.py migrate

# Создание администратора
python manage.py createsuperuser

# Запуск сервера
python manage.py runserver
