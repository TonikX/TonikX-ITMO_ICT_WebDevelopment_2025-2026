# Django REST Framework API Documentation

Добро пожаловать в документацию проекта **Django REST Framework API**! 

Этот проект демонстрирует использование Django REST Framework для создания RESTful API в рамках практического задания 3.2 по курсу "Веб-разработка" ИТМО.

## Цели проекта

- Изучение принципов REST API
- Практическое применение Django REST Framework
- Создание полнофункционального API для управления воинами, профессиями и навыками
- Демонстрация различных подходов к сериализации данных

## Основные возможности

- **RESTful API** - полное соответствие принципам REST
- **CRUD операции** - создание, чтение, обновление и удаление данных
- **Сериализация** - различные уровни детализации данных
- **Оптимизация** - эффективные запросы к базе данных
- **Валидация** - автоматическая проверка входных данных

## 🛠 Технологии

- **Django 4.2.7** - веб-фреймворк
- **Django REST Framework** - для создания REST API
- **SQLite** - база данных
- **MkDocs** - генерация документации
