# Табло победителей автогонок

## Описание

Сайт отображает информацию об участниках автогонок и их результатах.

## Отображаемые данные об участнике

* **ФИО участника**
* **Название команды**
* **Описание автомобиля**
* **Описание участника**
* **Опыт**
* **Класс участника**

## Функциональные требования

* **Регистрация новых пользователей**
* **Просмотр автогонок и регистрация гонщиков**

  * Пользователь может **редактировать** и **удалять** **только свои** регистрации
* **Отзывы и комментарии к автогонкам** (только для зарегистрированных пользователей)

  * При добавлении комментария сохраняются:

    * дата заезда
    * текст комментария
    * тип комментария: *вопрос о сотрудничестве*, *вопрос о гонках*, *иное*
    * рейтинг (1–10)
    * информация о комментаторе

## Администрирование

* Администратор указывает **время заезда** и **результат** через **Django Admin**

## Клиентская часть

* Формируется **таблица всех заездов и результатов** конкретной гонки

## Быстрый старт

```bash
# Клонирование репозитория
git clone <repository-url>
cd PythonProject2

# Создание виртуального окружения
python -m venv .venv
.venv\Scripts\activate  # Windows

# Установка зависимостей
pip install -r requirements.txt

# Применение миграций
python manage.py migrate

# Создание демо-данных
python manage.py create_demo_data

# Создание суперпользователя
python manage.py createsuperuser

# Запуск сервера
python manage.py runserver
```

Приложение будет доступно по адресу: [http://127.0.0.1:8000/](http://127.0.0.1:8000/)
