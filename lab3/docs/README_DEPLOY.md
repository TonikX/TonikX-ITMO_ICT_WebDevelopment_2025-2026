## 🔧 Активация GitHub Pages

Если сайт еще не доступен, выполните следующие шаги:

1. Перейдите на GitHub: https://github.com/Cherepnya-Yaroslav/web-itmo
2. Откройте **Settings** (Настройки)
3. В левом меню найдите **Pages**
4. В разделе **Source** выберите:
   - **Branch:** `gh-pages`
   - **Folder:** `/ (root)`
5. Нажмите **Save** (Сохранить)

После этого GitHub начнет публиковать ваш сайт.

## 📝 Обновление документации

После любых изменений в документации выполните:

```bash
cd lab3/docs
source ../venv/bin/activate
mkdocs gh-deploy
```

Или просто сделайте коммит и пуш - GitHub Actions автоматически обновит документацию (если настроен workflow).

## 🚀 Автоматическая публикация

Я создал GitHub Actions workflow (`.github/workflows/docs.yml`), который будет автоматически публиковать документацию при каждом коммите в ветку `main`, если изменены файлы в `lab3/docs/`.

## 📚 Локальный просмотр

Для локального просмотра документации:

```bash
cd lab3/docs
source ../venv/bin/activate
mkdocs serve
```

Документация будет доступна по адресу: `http://127.0.0.1:8000`
