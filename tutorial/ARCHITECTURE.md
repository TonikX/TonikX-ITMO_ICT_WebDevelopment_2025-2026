```

├─────────────────────────────────────────────────────────────┤
```

## Поток данных

1. **HTTP запрос** → URL Router
2. **URL Router** → View (контроллер)
3. **View** → Model (запрос к БД)
4. **Model** → Database (SQLite)
5. **Database** → Model (результат запроса)
6. **Model** → View (данные)
7. **View** → Template (рендеринг)
8. **Template** → HTTP Response (HTML)

## Компоненты системы

### Frontend
- HTML шаблоны с CSS стилями
- Адаптивный дизайн
- Таблицы для отображения данных

### Backend
- Django ORM для работы с БД
- Модели с валидацией данных
- Контроллеры с обработкой ошибок
- URL маршрутизация

### Database
- SQLite (встроенная БД)
- Миграции для управления схемой
- Связи между таблицами (FK)

### Admin Interface
- Автоматически генерируемый интерфейс
- Управление всеми моделями
- Фильтрация и поиск
