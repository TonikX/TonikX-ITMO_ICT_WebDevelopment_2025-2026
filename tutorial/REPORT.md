- state_number (CharField, max_length=10, с валидаторами)

- car (ForeignKey to Car)
- start_date (DateField)
- end_date (DateField, null=True, blank=True)

**DriverLicense (Водительское удостоверение):**
- owner (ForeignKey to Owner)
- license_number (CharField, max_length=10)
- license_type (CharField с choices: A, B, C, D)
- issue_date (DateField)

## Запуск проекта

1. Активировать виртуальное окружение:
   ```bash
   source tutorial-env/bin/activate
   ```

2. Запустить сервер:
   ```bash
   python manage.py runserver
   ```

3. Открыть в браузере:
   - Главная: http://127.0.0.1:8000/
   - Админка: http://127.0.0.1:8000/admin/
   - Владелец: http://127.0.0.1:8000/owner/1/
