# Django проект: Система управления автовладельцами

## Описание проекта

Этот проект представляет собой веб-приложение на Django для управления информацией об автовладельцах, их автомобилях и водительских удостоверениях.

## Структура проекта

- **django_project_tonikx** - основной проект Django
- **project_first_app** - приложение для управления автовладельцами
- **templates** - HTML шаблоны
- **tutorial-env** - виртуальное окружение Python

## Модели данных

### Owner (Владелец)
- first_name - Имя
- last_name - Фамилия  
- birth_date - Дата рождения

### Car (Автомобиль)
- brand - Марка
- model - Модель
- color - Цвет
- state_number - Государственный номер

### Ownership (Владение)
- owner - Владелец (внешний ключ)
- car - Автомобиль (внешний ключ)
- start_date - Дата начала владения
- end_date - Дата окончания владения (может быть пустой)

### DriverLicense (Водительское удостоверение)
- owner - Владелец (внешний ключ)
- license_number - Номер удостоверения
- license_type - Тип удостоверения (A, B, C, D)
- issue_date - Дата выдачи

## Установка и запуск

1. **Активация виртуального окружения:**
   ```bash
   source tutorial-env/bin/activate
   ```

2. **Запуск сервера разработки:**
   ```bash
   python manage.py runserver
   ```

3. **Доступ к приложению:**
   - Главная страница: http://127.0.0.1:8000/
   - Админ-панель: http://127.0.0.1:8000/admin/
   - Информация о владельце: http://127.0.0.1:8000/owner/1/

## Админ-панель

Для доступа к админ-панели используйте:
- Логин: admin
- Пароль: admin123

В админ-панели можно:
- Просматривать и редактировать владельцев
- Управлять автомобилями
- Настраивать записи о владении
- Работать с водительскими удостоверениями

## API Endpoints

- `GET /owner/<int:owner_id>/` - Получить информацию о владельце по ID
