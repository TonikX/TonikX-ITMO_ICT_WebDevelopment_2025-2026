# Инструкция по использованию Django приложения

## 🚀 Быстрый старт

### 1. Запуск сервера
```bash
cd django_project_tonikx
source ../tutorial-env/bin/activate
python manage.py runserver
```

### 2. Доступ к приложению
Откройте браузер и перейдите по адресу: **http://127.0.0.1:8000/**

## 📱 Основные страницы

### 🏠 Главная страница (`/`)
- **Описание**: Обзор системы с статистикой
- **Функции**:
  - Показывает количество владельцев, автомобилей, записей о владении
  - Список доступных владельцев с возможностью перехода к детальной информации
  - Ссылки на админ-панель и примеры

### 👤 Страница владельца (`/owner/<id>/`)
- **Описание**: Детальная информация о конкретном владельце
- **Функции**:
  - Личная информация владельца
  - Список водительских удостоверений
  - История владения автомобилями
- **Пример**: http://127.0.0.1:8000/owner/1/

### 🔧 Админ-панель (`/admin/`)
- **Описание**: Управление всеми данными системы
- **Доступ**: 
  - Логин: `admin`
  - Пароль: `admin123`
- **Функции**:
  - CRUD операции для всех моделей
  - Фильтрация и поиск
  - Управление пользователями

## 🛠️ Техническая информация

### Структура URL:
```
/                    - Главная страница
/admin/              - Админ-панель Django
/owner/<int:id>/     - Страница владельца по ID
```

### Модели данных:
- `Owner` - Владельцы автомобилей
- `Car` - Автомобили
- `Ownership` - История владения (связь многие-ко-многим)
- `DriverLicense` - Водительские удостоверения

### База данных:
- SQLite (файл: `db.sqlite3`)
- Автоматически создается при первом запуске
- Миграции применены

## ❓ Решение проблем

### Ошибка 404 на главной странице
- Убедитесь, что сервер запущен
- Проверьте URL: должен быть `http://127.0.0.1:8000/` (с слешем в конце)

### Ошибка подключения к базе данных
- Убедитесь, что миграции применены: `python manage.py migrate`
- Проверьте права доступа к файлу `db.sqlite3`

### Проблемы с админ-панелью
- Убедитесь, что создан суперпользователь: `python manage.py createsuperuser`
- Используйте правильные учетные данные: admin/admin123
