# Табло победителей автогонок

Веб-приложение для управления автогонками, регистрацией участников и отображением результатов.

## Описание

Проект представляет собой полнофункциональное веб-приложение на Django для организации автогонок:
- Регистрация и аутентификация пользователей
- Создание профилей водителей
- Просмотр гонок и заездов
- Регистрация на гонки
- Комментарии и отзывы
- Таблицы результатов с сортировкой
- Админ-панель для управления

## Требования

- Python 3.10+
- Django 5.1+
- SQLite (встроенная БД)
- Bootstrap 5 (через CDN)

## Установка и запуск

### Windows (PowerShell)

```powershell
# Создание виртуального окружения
python -m venv venv

# Активация виртуального окружения
.\venv\Scripts\Activate.ps1

# Установка зависимостей
pip install -r requirements.txt

# Создать миграции
python manage.py makemigrations racing

# Применить миграции
python manage.py migrate

# Создание суперпользователя
python manage.py createsuperuser

# Создание демо-данных (опционально)
python manage.py create_demo_data

# Запуск сервера
python manage.py runserver
```

### Linux/macOS

```bash
# Создание виртуального окружения
python3 -m venv venv

# Активация виртуального окружения
source venv/bin/activate

# Установка зависимостей
pip install -r requirements.txt

# Применение миграций
python manage.py migrate

# Создание суперпользователя
python manage.py createsuperuser

# Создание демо-данных (опционально)
python manage.py create_demo_data

# Запуск сервера
python manage.py runserver
```

После запуска приложение будет доступно по адресу: http://127.0.0.1:8000/

## Настройки

### Язык и часовой пояс

В файле `racing_project/settings.py`:

```python
LANGUAGE_CODE = 'ru'  # Язык интерфейса
TIME_ZONE = 'Europe/Moscow'  # Часовой пояс
```

Доступные часовые пояса:
- `Europe/Moscow` - Москва (UTC+3)
- `Europe/Kaliningrad` - Калининград (UTC+2)
- `Asia/Yekaterinburg` - Екатеринбург (UTC+5)
- `Asia/Novosibirsk` - Новосибирск (UTC+7)
- `Asia/Vladivostok` - Владивосток (UTC+10)

### База данных

По умолчанию используется SQLite. Файл БД: `db.sqlite3`
