# Цель и задачи лабораторной работы

## Цель работы

Целью лабораторной работы являлась разработка веб-приложения для управления птицефабрикой с использованием клиент-серверной архитектуры.

## Задачи работы

В рамках лабораторной работы были поставлены и выполнены следующие задачи:

* спроектировать REST API для предметной области;
* реализовать backend на FastAPI;
* реализовать frontend на React;
* настроить аутентификацию и авторизацию пользователей;
* реализовать CRUD-операции для основных сущностей;
* обеспечить связь между сущностями;
* реализовать аналитику и бизнес-логику;
* протестировать работу системы через пользовательский интерфейс.
