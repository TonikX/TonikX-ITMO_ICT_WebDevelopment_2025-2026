# Интерфейс цехов (Workshops)

## Назначение

Интерфейс предназначен для управления цехами птицефабрики.

## Реализованный функционал

* просмотр списка цехов;
* создание и редактирование цехов;
* удаление цехов;
* просмотр цеха вместе со всеми клетками.

## Использование

Цех объединяет набор клеток и используется для логической организации производства.
