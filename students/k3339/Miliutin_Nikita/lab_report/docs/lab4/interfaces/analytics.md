# Интерфейс аналитики (Analytics)

## Назначение

Интерфейс предназначен для анализа данных птицефабрики.

## Реализованный функционал

* анализ количества яиц по курицам;
* определение лучшего цеха по породе;
* расчет яиц на сотрудника в день;
* распределение кур по породам и цехам;
* анализ продуктивности пород.

## Роль в системе

Используется для принятия управленческих решений.
