# Интерфейс диет (Diets)

## Назначение

Интерфейс отвечает за управление рационом питания кур.

## Реализованный функционал

* просмотр списка диет;
* добавление новой диеты;
* редактирование состава;
* удаление диеты;
* просмотр детальной информации.

## Роль в системе

Диеты применяются к породам в зависимости от сезона.
