# Интерфейс истории перемещений кур (ChickenMove)

## Назначение

Интерфейс отвечает за хранение и просмотр истории перемещений кур между клетками.

## Реализованный функционал

* просмотр списка перемещений;
* фильтрация по курице и клеткам;
* детальный просмотр перемещения;
* удаление записей.

## Использование

Позволяет вести учет и аудит перемещений кур.
