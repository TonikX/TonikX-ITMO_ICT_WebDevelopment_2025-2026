# Интерфейс авторизации (Auth)

## Назначение

Интерфейс авторизации отвечает за управление учетными записями пользователей и контроль доступа к системе.

## Реализованный функционал

* регистрация нового пользователя;
* вход в систему с получением JWT-токена;
* получение данных текущего пользователя;
* изменение данных профиля.

## Использование

Все остальные интерфейсы системы доступны только авторизованным пользователям. JWT-токен используется для подтверждения прав доступа при каждом запросе к API.
