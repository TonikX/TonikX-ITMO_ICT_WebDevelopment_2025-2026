# Интерфейс пород (Breeds)

## Назначение

Интерфейс предназначен для управления породами кур.

## Реализованный функционал

* просмотр списка пород;
* фильтрация по характеристикам;
* создание и редактирование пород;
* удаление пород;
* просмотр пород с сезонными диетами.

## Использование

Породы используются для аналитики и подбора диет.
