# Интерфейс клеток (Cages)

## Назначение

Интерфейс отвечает за управление клетками, в которых содержатся куры.

## Реализованный функционал

* просмотр списка клеток;
* фильтрация по цеху, ряду и номеру;
* просмотр кур в клетке;
* просмотр закрепленных сотрудников;
* создание, редактирование и удаление клеток.

## Роль в системе

Клетки являются базовой единицей размещения кур внутри цехов.
