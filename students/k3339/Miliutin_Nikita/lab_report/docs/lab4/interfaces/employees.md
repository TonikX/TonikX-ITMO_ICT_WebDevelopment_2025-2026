# Интерфейс сотрудников (Employees)

## Назначение

Интерфейс отвечает за учет сотрудников птицефабрики.

## Реализованный функционал

* просмотр списка сотрудников;
* добавление нового сотрудника;
* редактирование данных;
* удаление сотрудника;
* просмотр назначений сотрудника на клетки.

## Особенности

Сотрудники могут быть закреплены за несколькими клетками в разные периоды времени.
