# Интерфейс диет по породам и сезонам (BreedDietSeason)

## Назначение

Интерфейс определяет, какая диета применяется к конкретной породе в определенный сезон.

## Реализованный функционал

* назначение диеты породе на сезон;
* обновление сезонной диеты;
* удаление назначения;
* upsert-интерфейс для удобного управления.

## Использование

Позволяет гибко управлять питанием кур в течение года.
