# Общее описание проекта

Разработанное веб-приложение предназначено для автоматизации управления птицефабрикой. Система позволяет вести учет кур, клеток, цехов, сотрудников, пород и диет, а также анализировать производственные показатели.

Проект охватывает основные бизнес-процессы птицефабрики и демонстрирует применение современных веб-технологий.
