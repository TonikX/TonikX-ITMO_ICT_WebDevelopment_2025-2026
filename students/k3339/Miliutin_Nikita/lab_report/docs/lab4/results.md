# Результаты выполнения лабораторной работы

В результате выполнения лабораторной работы было разработано полнофункциональное веб-приложение со следующими возможностями:

* реализован backend с REST API;
* реализован frontend пользовательский интерфейс;
* настроена JWT-аутентификация;
* реализован полный CRUD для основных сущностей;
* обеспечена связность данных между объектами;
* реализована аналитика и отчеты;
* обеспечена логическая целостность данных.
