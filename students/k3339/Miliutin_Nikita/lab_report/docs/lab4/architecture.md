# Архитектура и технологии проекта

## Общая архитектура

Проект реализован по клиент-серверной архитектуре и состоит из двух основных частей: backend и frontend.

## Backend

Backend реализован с использованием фреймворка FastAPI и отвечает за:

* бизнес-логику приложения;
* работу с базой данных;
* обработку HTTP-запросов;
* аутентификацию и авторизацию пользователей;
* предоставление REST API.

## Frontend

Frontend реализован с использованием библиотеки React и отвечает за:

* пользовательский интерфейс;
* навигацию между страницами;
* взаимодействие с backend через HTTP-запросы;
* отображение данных пользователю.

## Авторизация

Для защиты интерфейсов используется JWT-аутентификация. После успешного входа пользователь получает токен, который передается в заголовках запросов.
