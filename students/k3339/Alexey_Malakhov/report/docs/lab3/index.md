# Отчёт по лабораторной работе №3

**Студент**: Малахов Алексей Витальевич

**Вариант**: Подписочный хостинг премиум конента (аналог Boosty)

---

**Краткое описание проекта:** аналог сервиса Boosty, предназначенное для поддержки авторов. Платформа позволяет авторам публиковать контент, а пользователям — оформлять подписки, получать доступ к эксклюзивным материалам, писать комментарие и ставить лайки.

**Используемые технологии:** fastapi, sqlalchemy, starlette-admin, fastapi-authx, passlib, postgresql, alembic, pydantic, uvicorn.

В отчёте представлены разделы, посвящённые установке и запуску проекта, а также раздел с детальным описанием всех API-ручек, где указаны эндпоинты, методы запросов, параметры и примеры ответов для удобной интеграции в разные системы.

Все разделы отчёта доступны через навигацию слева.
