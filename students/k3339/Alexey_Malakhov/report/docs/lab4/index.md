## Краткое описание

Фронтенд-часть проекта представляет собой React приложение, которое обеспечивает взаимодействие пользователей с платформой поддержки авторов. Приложение позволяет пользователям регистрироваться, входить в систему, просматривать посты авторов, подписываться на них, оставлять комментарии и лайки. Авторы могут создавать профили, публиковать платные и бесплатные посты с медиаконтентом.

**Используемые технологии фронтенда:**

- **React 19** — основной фреймворк
- **TypeScript** — типизация
- **Vite** — сборщик проекта
- **React Router** — маршрутизация
- **TanStack Query** — управление состоянием и кэшированием данных
- **Tailwind CSS** — стилизация
- **Shadcn** — Библиотека компонентов
- **Axios** — HTTP клиент
- **Zod** — валидация данных

## Структура приложения

Фронтенд состоит из следующих ключевых частей:

### Страницы (Pages)

- **Main** — главная страница с лентой постов
- **LogIn** — страница входа в систему
- **SignUp** — страница регистрации
- **Profile** — профиль пользователя (личный кабинет)

### Компоненты (Components)

- **Layout** — компоненты макета (навигация, боковое меню)
- **Blocks** — блоки для отображения контента (посты, авторы)
- **Custom** — кастомные компоненты
- **Forms** — формы входа и регистрации
- **UI** — базовые UI компоненты (кнопки, инпуты, модалки)

### API интеграция

- **Axios конфигурация** — настройка HTTP клиента
- **React Query** — кэширование и управление запросами

Все разделы отчёта доступны через навигацию слева.
