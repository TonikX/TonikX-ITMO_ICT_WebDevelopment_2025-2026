    Лабораторная работа 3 - API Endpoints¶

    mark/top/¶

    Получить самый популярный тип самолета.

    Возвращает тип самолета, который чаще всего встречается в базе данных.

    Метод: GET

    Параметры: Нет

    Ответ: - 200: Успешно получен самый популярный тип - 500: Внутренняя ошибка сервера

    mark/all/¶

    Получить все типы самолетов и их количество.

    Возвращает список всех типов самолетов вместе с количеством самолетов каждого типа и общее количество самолетов.

    Метод: GET

    Параметры: Нет

    Ответ: - 200: Успешно получены все типы и количество - 500: Внутренняя ошибка сервера

    routes/pick/¶

    Получить рейсы с заполняемостью ниже заданного порога.

    Принимает необязательный параметр 'filled_less_than' (вещественное число от 0 до 1) для фильтрации рейсов по коэффициенту заполняемости.

    Метод: POST

    Параметры: - filled_less_than (float, опционально): Порог заполняемости (от 0.0 до 1.0)

    Ответ: - 200: Успешно получены рейсы - 400: Некорректные данные запроса - 500: Внутренняя ошибка сервера

    flights/{id}/available_seats/¶

    Получить доступные места для конкретного рейса.

    Возвращает список номеров доступных мест для указанного ID рейса.

    Метод: GET

    Параметры: - id (int): ID рейса

    Ответ: - 200: Успешно получены доступные места - 404: Рейс не найден - 500: Внутренняя ошибка сервера

    planes/in_repair/¶

    Получить все самолеты, находящиеся в ремонте.

    Возвращает список самолетов со статусом 'repair'.

    Метод: GET

    Параметры: Нет

    Ответ: - 200: Успешно получены самолеты в ремонте - 500: Внутренняя ошибка сервера

    employees/count/¶

    Получить общее количество сотрудников (членов экипажа).

    Возвращает общее количество членов экипажа в базе данных.

    Метод: GET

    Параметры: Нет

    Ответ: - 200: Успешно получено количество сотрудников - 500: Внутренняя ошибка сервера

    flights/full/¶

    Получить полную информацию о всех рейсах.

    Возвращает список всех рейсов с полной информацией о связанных объектах.

    Метод: GET

    Параметры: Нет

    Ответ: - 200: Успешно получены рейсы

    planes_with_flights/¶

    Получить самолеты с информацией о рейсах.

    Возвращает список самолетов с информацией о связанных рейсах.

    Метод: GET

    Параметры: Нет

    Ответ: - 200: Успешно получены самолеты и рейсы

    tickets_with_passengers/¶

    Получить билеты с информацией о пассажирах.

    Возвращает список билетов с информацией о пассажирах, местах и рейсах.

    Метод: GET

    Параметры: Нет

    Ответ: - 200: Успешно получены билеты с пассажирами

    avg_seats_per_flight/¶

    Среднее количество мест на рейс.

    Возвращает среднее количество мест на один рейс.

    Метод: GET

    Параметры: Нет

    Ответ: - 200: Успешно получено среднее количество мест

    most_popular_route/¶

    Самый популярный маршрут.

    Возвращает маршрут (по количеству рейсов), который используется чаще всего.

    Метод: GET

    Параметры: Нет

    Ответ: - 200: Успешно получен самый популярный маршрут - 204: Нет данных о рейсах

    tickets_per_flight/¶

    Количество билетов на каждый рейс.

    Возвращает количество проданных билетов для каждого рейса.

    Метод: GET

    Параметры: Нет

    Ответ: - 200: Успешно получено количество билетов на рейс

    flights_occupancy/¶

    Заполняемость рейсов.

    Возвращает отношение купленных мест к общему количеству мест для каждого рейса.

    Метод: GET

    Параметры: Нет

    Ответ: - 200: Успешно получена заполняемость рейсов
