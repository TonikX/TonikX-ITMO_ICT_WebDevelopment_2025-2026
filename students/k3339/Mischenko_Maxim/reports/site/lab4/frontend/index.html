    Фронтенд приложение¶

    Общая информация¶

    Фронтенд приложения реализован с использованием Vue.js 3 и Vuetify для создания современного пользовательского интерфейса. Приложение предоставляет интерактивные инструменты для управления авиакомпанией и аналитики.

    Основные компоненты¶

    Главная страница (Dashboard)¶

    Расположение: /src/views/Dashboard.vue

    Предоставляет централизованную панель управления с тремя основными разделами: 1. Список самолетов, находящихся в ремонте 2. Инструмент для анализа маршрутов с низкой заполняемостью 3. Поиск доступных мест на рейсе

    Страница самолетов (Planes)¶

    Расположение: /src/views/Planes.vue

    Отображает список всех самолетов, находящихся в ремонте, с детальной информацией: - ID борта - Модель самолета - Авиакомпания-владелец - Технические характеристики - Статус (всегда "REPAIR")

    Особенности: - Современный дизайн с использованием стекла морфизма (glassmorphism) - Интерактивная таблица с возможностью обновления данных - Визуальная индикация статуса

    Страница маршрутов (Routes)¶

    Расположение: /src/views/Routes.vue

    Предоставляет инструмент для анализа эффективности маршрутов: - Слайдер для установки порога заполняемости - Список рейсов с коэффициентом заполняемости ниже порога - Визуальная индикация рейсов с низкой заполняемостью

    Особенности: - Интуитивный интерфейс с разделением на панель фильтров и панель результатов - Анимированные переходы - Визуальная обратная связь при взаимодействии

    Страница мест (Seats)¶

    Расположение: /src/views/Seats.vue

    Предоставляет визуальный интерфейс для поиска доступных мест на рейсе: - Поле ввода для указания ID рейса - Визуализация доступных мест в виде схемы салона самолета - Индикация количества доступных мест

    Особенности: - 3D-подобная визуализация самолета - Интерактивные элементы с анимацией - Адаптивный дизайн

    Страницы аутентификации¶

    • Login.vue - форма входа в систему
    • Register.vue - форма регистрации
    • Profile.vue - страница профиля пользователя

    Навигация¶

    Навигация осуществляется через верхнюю панель приложения: - Ремонт - переход к странице самолетов в ремонте - Маршруты - переход к странице анализа маршрутов - Места - переход к странице поиска мест - Профиль - переход к странице профиля

    Технические особенности¶

    API клиент¶

    Расположение: /src/api.js

    Использует Axios для взаимодействия с бэкенд API: - Базовый URL: http://127.0.0.1:8000 - Автоматическое добавление токена авторизации к запросам

    Роутинг¶

    Расположение: /src/router/index.js

    Использует Vue Router для навигации между страницами: - Защита маршрутов с проверкой аутентификации - Динамическая навигация

    Компоненты¶

    • Использование Vuetify для создания современного UI
    • Адаптивный дизайн для различных устройств
    • Анимации и переходы для улучшения пользовательского опыта

    Стилизация¶

    Приложение использует современные подходы к стилизации: - Glassmorphism - эффект стекла для карточек - Градиенты для визуального выделения элементов - Анимации при наведении и взаимодействии - Темная цветовая схема для уменьшения усталости глаз

    Адаптивность¶

    Приложение адаптировано для использования на различных устройствах: - Десктопные браузеры - Планшеты - Мобильные устройства

    Используются адаптивные компоненты Vuetify и медиа-запросы для оптимизации отображения на разных экранах.
