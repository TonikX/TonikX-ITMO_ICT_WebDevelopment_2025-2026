# Использование

## Маршруты проекта

| URL                  | Назначение                                      |
|----------------------|-------------------------------------------------|
| `/`                  | Главная страница с заданиями                    |
| `/login/`            | Вход                                            |
| `/logout/`           | Выход                                           |
| `/register/`         | Регистрация                                     |
| `/assignments/`      | Список заданий                                  |
| `/assignments/<id>/` | Детали задания                                  |
| `/grades/`           | Таблица оценок                                  |

## Функциональность
- Пользователи могут регистрироваться и входить
- Учителя могут ставить оценки через админку
- Таблица оценок доступна всем пользователям
