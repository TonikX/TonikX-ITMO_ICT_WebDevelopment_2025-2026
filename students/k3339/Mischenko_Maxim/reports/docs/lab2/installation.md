# Установка

## Зависимости
- Python 3.12+
- Django 3.x
- PostgreSQL
- MkDocs

```py
python -m venv .venv
source .venv/bin/activate
```

```py
pip install -r requirements.txt
```

```py
python manage.py migrate
```

```py
python manage.py createsuperuser
```

```py
python manage.py runserver
```
