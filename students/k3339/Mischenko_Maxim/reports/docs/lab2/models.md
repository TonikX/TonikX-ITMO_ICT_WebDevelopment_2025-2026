# Модели данных

## Subject
- **name** — название предмета
- **description** — описание предмета

## Assignment
- **subject** — предмет
- **title** — название задания
- **description** — описание задания
- **teacher** — преподаватель
- **date_issued** — дата выдачи
- **period_days** — период выполнения
- **penalties_info** — информация о штрафах

## Submission
- **assignment** — задание
- **student** — ученик
- **text** — текст работы
- **submitted_at** — дата сдачи
- **grade** — оценка
- **graded_by** — кто поставил оценку
- **feedback** — комментарий
