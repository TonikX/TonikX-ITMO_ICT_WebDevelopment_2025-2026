# Лабораторная работа 4: Веб-приложение для авиакомпании

## Цель работы

Целью данной лабораторной работы является разработка полнофункционального веб-приложения для авиакомпании с использованием Django REST Framework для бэкенда и Vue.js для фронтенда. Приложение включает в себя систему управления авиаперевозками, включающую информацию о самолетах, рейсах, пассажирах, билетах и экипаже.

## Основные компоненты

### Бэкенд (Django REST Framework)
- Модели данных для представления всех сущностей авиакомпании
- REST API для управления данными
- Аутентификация пользователей
- Специальные эндпоинты для аналитики и отчетности

### Фронтенд (Vue.js)
- Интерактивный пользовательский интерфейс
- Страницы для управления самолетами, маршрутами и местами
- Панель администратора с аналитическими инструментами
- Аутентификация и авторизация пользователей

## Архитектура приложения

Приложение состоит из двух основных частей:
1. **airline_admin** - Django приложение, реализующее бэкенд API
2. **airline_client** - Vue.js приложение, реализующее фронтенд

Компоненты связаны через REST API, что позволяет легко масштабировать и модифицировать приложение.

## Развертывание

Приложение использует Docker Compose для упрощения развертывания:
- PostgreSQL база данных
- Django приложение для бэкенда
- Vue.js приложение для фронтенда

## Функциональность

### Управление самолетами
- Просмотр всех самолетов
- Просмотр самолетов, находящихся в ремонте
- Информация о техническом состоянии

### Управление маршрутами
- Анализ заполняемости рейсов
- Поиск рейсов с низкой заполняемостью
- Планирование маршрутов

### Управление местами
- Просмотр доступных мест на рейсе
- Визуализация расположения мест в салоне

### Аналитика
- Статистика по популярным маршрутам
- Анализ загрузки самолетов
- Отчеты по эффективности перевозок
