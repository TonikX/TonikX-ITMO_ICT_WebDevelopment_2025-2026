# API Endpoints

## Общая информация

API реализован с использованием Django REST Framework и предоставляет эндпоинты для управления данными авиакомпании и получения аналитической информации.

Базовый URL: `http://localhost:8000/air/`

## Эндпоинты

### Получение самой популярной модели самолета
```
GET /mark/top/
```

Возвращает самую популярную модель самолета по количеству экземпляров в базе данных.

Ответ:
```json
{
  "top_mark": "Boeing 737"
}
```

### Получение списка всех моделей самолетов
```
GET /mark/all/
```

Возвращает список всех моделей самолетов с количеством экземпляров каждой модели и общее количество самолетов.

Ответ:
```json
{
  "marks": [
    {
      "mark": "Boeing 737",
      "count_boards": 5
    },
    {
      "mark": "Airbus A320",
      "count_boards": 3
    }
  ],
  "count_boards": 8
}
```

### Поиск рейсов с низкой заполняемостью
```
POST /routes/pick/
```

Возвращает список рейсов с коэффициентом заполняемости ниже заданного порога.

Параметры запроса:
```json
{
  "filled_less_than": 0.5
}
```

Ответ:
```json
{
  "flights": [
    {
      "id": 1,
      "plane": 1,
      "status": "scheduled",
      "departure_airport": "SVO",
      "destination_airport": "LED",
      "arrival_time": "2023-12-20T12:00:00Z",
      "arrival_time_fact": "2023-12-20T12:00:00Z",
      "departure_time": "2023-12-20T10:00:00Z",
      "departure_time_fact": "2023-12-20T10:00:00Z"
    }
  ]
}
```

### Получение доступных мест на рейсе
```
GET /flights/{id}/available_seats/
```

Возвращает список доступных мест на указанном рейсе.

Параметры:
- `id` - ID рейса

Ответ:
```json
{
  "available_seats": ["1A", "1B", "2A", "2B"]
}
```

### Получение самолетов, находящихся в ремонте
```
GET /planes/in_repair/
```

Возвращает список всех самолетов со статусом "repair".

Ответ:
```json
{
  "planes": [
    {
      "id": 1,
      "company": 1,
      "status": "repair",
      "flight_duration": 360,
      "mark": "Boeing 737",
      "last_technical_service": "2023-12-01T10:00:00Z"
    }
  ]
}
```

### Получение количества сотрудников
```
GET /employees/count/
```

Возвращает общее количество сотрудников (членов экипажа).

Ответ:
```json
{
  "employees_count": 25
}
```

## Аутентификация

API использует токенную аутентификацию. Для доступа к защищенным эндпоинтам необходимо включить заголовок авторизации:

```
Authorization: Token your_token_here
```
