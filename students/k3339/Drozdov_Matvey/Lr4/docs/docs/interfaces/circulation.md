# Страница выдачи книг (Circulation)

---

## Общая структура

Страница состоит из двух блоков:

1. Выдача книги
2. Активные выдачи

---

## Блок выдачи

### Поиск читателя

Поддерживает поиск по:

- ФИО
- телефону
- паспорту
- номеру билета

Выбор из выпадающего списка.

---

### Поиск книги

Поддерживает поиск по:

- названию
- автору
- шифру

---

### Выбор зала

Выбирается из списка доступных залов.

---

### Количество экземпляров

Числовое поле.

---

### Дата выдачи

По умолчанию — текущая дата.

---

## Проверка доступности

Перед выдачей отображается:

- всего экземпляров в зале
- на руках
- доступно

Алгоритм:

1. Получение BookStock
2. Вычитание активных Loan

---

## Кнопка "Выдать"

Отправляет POST запрос.

Backend проверяет:

- наличие билета
- закрепление за залом
- доступность экземпляров

---

## Активные выдачи

Отображает:

- читателя
- книгу
- зал
- дату выдачи
- количество

---

### Действие "Принять"

PATCH запрос:

устанавливает returned_at.
