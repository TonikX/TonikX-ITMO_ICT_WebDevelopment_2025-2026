# Страница входа (Login)

## Структура интерфейса

Страница содержит форму авторизации с минималистичным интерфейсом.

### Поля формы

#### Логин (username)

- текстовое поле
- обязательное
- используется для идентификации пользователя

#### Пароль (password)

- поле типа password
- скрывает вводимые символы
- обязательное

---

## Кнопки управления

### Войти

Назначение:

- отправляет данные формы на сервер
- блокируется во время выполнения запроса (loading state)

Состояния:

- Войти — стандартное состояние
- Вхожу... — во время выполнения запроса

---

## Обработка ошибок

При неудачной авторизации отображается сообщение:

Не удалось войти. Проверь логин/пароль

Ошибки выводятся под кнопкой входа.

---

## Навигация

### Ссылка "Нет аккаунта? Регистрация"

Перенаправляет пользователя на страницу регистрации.

---

## Поведение интерфейса

### Успешный вход

1. Пользователь вводит логин и пароль.
2. Отправляется запрос авторизации.
3. Сохраняется состояние пользователя (auth store).
4. Выполняется переход на главную страницу.

---

### Ошибка входа

1. Пользователь вводит неверные данные.
2. Сервер возвращает ошибку.
3. Интерфейс отображает сообщение об ошибке.
4. Пользователь может повторить попытку.
