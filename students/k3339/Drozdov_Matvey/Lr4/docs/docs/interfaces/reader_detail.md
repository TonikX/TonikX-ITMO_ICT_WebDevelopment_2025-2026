# Страница читателя (Reader Detail)

## Назначение

Карточка читателя отображает:

- персональные данные
- активный билет
- текущее членство
- текущий зал
- историю изменений

Страница является центральной точкой управления читателем.

---

## Общая структура

Страница разделена на блоки:

1. Паспорт читателя
2. Членство
3. История залов
4. Дополнительная информация

---

## Паспорт читателя

Отображает:

- ФИО
- телефон
- паспорт
- дату рождения
- образование
- наличие учёной степени

---

## Активный билет

Определяется автоматически:

- выбирается запись ticket_history
- где valid_to == null

---

## Членство

Отображает:

- текущий статус (активен / не активен)
- последнюю дату регистрации

Членство определяется по истории membership events.

---

## Текущий зал

Определяется через ReaderHallHistory:

- запись valid_to == null считается текущей.

---

## История залов

Отображает:

- список всех залов
- даты начала и окончания закрепления

---

## Навигация

### Кнопка "Назад"

Возвращает пользователя к списку читателей.

---

## Состояния интерфейса

### Загрузка

- блокируются действия
- отображается статус загрузки

---

### Ошибка

Показывается сообщение ошибки загрузки данных.

---

## UX особенности

- вся ключевая информация отображается без переходов между страницами
- текущие значения вычисляются автоматически
- история отображается отдельно от текущего состояния
