# Страница отчётов (Reports)


## Доступные отчёты

### Активные книги читателя

Отображает:

- текущие выдачи выбранного читателя
- книгу
- зал
- дату выдачи

---

### Просроченные выдачи

Показывает:

- выдачи старше указанного количества дней
- читателя
- книгу
- зал

Позволяет быстро находить проблемные записи.

---

### Редкие книги

Показывает активные выдачи книг, у которых:

суммарное количество экземпляров <= 2.

---

### Статистика читателей

Отображает:

- количество читателей младше 20 лет
- распределение по образованию
- процент наличия учёной степени

---

### Очистка старых читателей

Кнопка запуска административного действия:

Удаляет читателей:

- зарегистрированных более года назад
- без перерегистрации

---

### Месячный отчёт

Отображает:

- ежедневное количество экземпляров по залам
- количество читателей по залам
- суммарные показатели
- количество новых записей за месяц

---

## Структура интерфейса

- вкладки или карточки отчётов
- таблицы данных
- сводные показатели
