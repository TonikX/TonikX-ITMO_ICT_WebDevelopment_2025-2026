# Страница читальных залов (Halls)
---

## Общая структура интерфейса

Страница состоит из:

- заголовка
- кнопки перехода на главную
- панели управления
- таблицы залов
- сообщений состояния

---

## Панель управления

### Поиск

Позволяет фильтровать залы по:

- номеру
- названию

Фильтрация выполняется на frontend.

---

### Кнопка "Обновить"

Перезагружает:

- список залов
- количество читателей
- количество экземпляров

---

### Кнопка "+ Добавить зал"

Открывает страницу создания зала.

---

## Таблица залов

Отображает:

- номер строки
- номер зала
- название
- вместимость
- количество закреплённых читателей
- количество экземпляров книг

Название зала является ссылкой на страницу деталей.

---

## Подсчёт читателей

Алгоритм:

- используется ReaderHallHistory
- учитываются записи:

valid_to == null

---

## Подсчёт экземпляров

Алгоритм:

- используется BookStock
- суммируются copies по hall_id

---

## Действия

### Открыть

Переход:

/halls/{id}

---

### Удалить

Поведение:

1. confirm диалог
2. DELETE запрос
3. удаление строки без полной перезагрузки

---

## Состояния

### Загрузка

- блокировка кнопки обновления

### Ошибка

Отображается сообщение ошибки.

### Пустой список

Отображается сообщение:

"Залов пока нет".
