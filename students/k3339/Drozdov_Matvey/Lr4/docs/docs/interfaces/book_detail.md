# Страница книги (Book Detail)

## Общая структура

Страница разделена на три основные секции:

1. Паспорт книги
2. История шифров
3. Фонд по залам

---

## Паспорт книги

Отображает:

- название
- авторов
- издательство
- год публикации
- раздел
- текущий шифр

---

### Кнопки

#### Назад

Возвращает пользователя к списку книг.

#### Обновить

Перезагружает все данные страницы.

#### Удалить

Удаляет книгу.

Поведение:

1. confirm диалог
2. DELETE запрос
3. редирект на /books

---

## История шифров

Таблица отображает:

- шифр
- дату начала действия
- дату окончания
- статус (действует/закрыт)

---

### Добавление нового шифра

Алгоритм:

1. Закрывается текущий активный шифр (valid_to = today)
2. Создаётся новый шифр
3. Список обновляется

---

## Фонд по залам

Отображает количество экземпляров книги по каждому залу.

Поля:

- зал
- количество экземпляров

---

### Управление фондом

Можно:

- выбрать зал
- указать количество экземпляров
- сохранить изменения

Логика:

- если запись существует — PATCH
- иначе — POST

---

## Вычисляемые значения

### Текущий шифр

Определяется как запись с:

valid_to == null

---

### Всего экземпляров

Сумма copies по всем залам.

---

## Состояния интерфейса

### Загрузка

- блокируются кнопки
- отображается статус загрузки

---

### Ошибка

Показывается сообщение с текстом ошибки.
