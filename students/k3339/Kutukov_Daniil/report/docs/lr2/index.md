# Лабораторная работа 2: Django проекты

Добро пожаловать в документацию по лабораторной работе №2 курса "Web-программирование"!

## 📋 Описание

В рамках данной лабораторной работы реализованы два полноценных веб-приложения на Django:

1. **Доска домашних заданий (Homework Board)** - система управления домашними заданиями для учебных заведений
2. **Система управления автовладельцами (Car Owners Management)** - приложение для учета автомобилей и их владельцев

## 🎯 Цели работы

- Изучение основ Django framework
- Работа с моделями данных и ORM
- Создание CRUD интерфейсов
- Работа с формами и валидацией
- Реализация аутентификации и авторизации
- Применение классовых и функциональных представлений
- Работа с шаблонами Django

## 🚀 Проекты

### 1. Доска домашних заданий

Современная веб-система для управления домашними заданиями в учебных заведениях.

**Ключевые возможности:**

- ✅ Управление заданиями (создание, редактирование, удаление)
- 👥 Разделение ролей (студенты, преподаватели, администраторы)
- 📝 Сдача заданий студентами
- 📊 Система оценивания и обратной связи
- 🔍 Поиск и фильтрация заданий
- 📈 Статистика успеваемости
- 🎨 Адаптивный дизайн с Bootstrap 5

[Подробнее о проекте →](homework_board/index.md)

### 2. Система управления автовладельцами

Веб-приложение для управления информацией об автовладельцах, их автомобилях и водительских удостоверениях.

**Ключевые возможности:**

- 🚗 Управление автомобилями
- 👤 Управление владельцами
- 📋 Отслеживание истории владения
- 🪪 Управление водительскими удостоверениями
- 🔐 Система аутентификации
- 📱 Адаптивный интерфейс

[Подробнее о проекте →](tutorial/index.md)

## 💻 Технологический стек

=== "Backend"
    - **Framework**: Django 5.2.7
    - **Язык**: Python 3.12+
    - **База данных**: SQLite
    - **ORM**: Django ORM

=== "Frontend"
    - **CSS Framework**: Bootstrap 5
    - **HTML**: HTML5
    - **CSS**: CSS3
    - **JavaScript**: Vanilla JS

=== "Инструменты"

## 🎓 Учебные материалы

В процессе выполнения лабораторной работы были изучены следующие темы:

1. **Django основы**
   - Структура Django проекта
   - Приложения и их взаимодействие
   - Система настроек

2. **Модели и база данных**
   - Определение моделей
   - Миграции базы данных
   - Связи между моделями (OneToOne, ForeignKey, ManyToMany)
   - Django ORM

3. **Представления**
   - Функциональные представления (Function-Based Views)
   - Классовые представления (Class-Based Views)
   - Generic Views (ListView, DetailView, CreateView, UpdateView, DeleteView)

4. **Формы**
   - ModelForm
   - Валидация данных
   - Обработка файлов

5. **Шаблоны**
   - Django Template Language
   - Наследование шаблонов
   - Статические файлы
   - Интеграция с Bootstrap

6. **Аутентификация**
   - Система пользователей Django
   - Расширение модели User
   - Регистрация и авторизация
   - Разграничение доступа
