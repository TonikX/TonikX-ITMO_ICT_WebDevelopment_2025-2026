# Система управления автовладельцами

## Обзор

Веб-приложение для управления информацией об автовладельцах, их автомобилях и водительских удостоверениях.

## Модели данных

### User (расширенная модель пользователя)
- Стандартные поля Django User

### Car (автомобиль)
- Марка, модель, цвет, госномер, VIN

### Ownership (владение)
- Владелец, автомобиль, даты начала и окончания владения

### DriverLicense (водительское удостоверение)
- Владелец, номер, тип (A, B, C, D), дата выдачи

## Представления

- CRUD для автомобилей
- CRUD для владельцев
- Просмотр истории владения
- Управление удостоверениями

## Формы

- Формы для создания/редактирования автомобилей и владельцев

## URL маршруты

- `/cars/` - список автомобилей
- `/owners/` - список владельцев
- `/licenses/` - список удостоверений

## Шаблоны

Используют Bootstrap для стилизации.
