# Представления Tutorial

## Классовые представления

- `CarListView` - список автомобилей
- `CarDetailView` - детализация автомобиля
- `CarCreateView` - создание автомобиля
- `CarUpdateView` - редактирование автомобиля
- `CarDeleteView` - удаление автомобиля

Аналогично для Owner и DriverLicense.

## Аутентификация

Используется стандартная Django аутентификация.
