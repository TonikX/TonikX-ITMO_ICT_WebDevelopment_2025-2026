# Шаблоны Homework Board

## Базовый шаблон

`templates/base.html` с Bootstrap 5.

## Шаблоны для заданий

- `assignment_list.html` - список заданий
- `assignment_detail.html` - детализация задания
- `assignment_form.html` - форма создания/редактирования
- `assignment_confirm_delete.html` - подтверждение удаления

## Шаблоны для сдач и оценок

- `submission_form.html` - форма сдачи
- `grade_form.html` - форма оценки

Все шаблоны наследуются от base.html и используют Bootstrap компоненты.
