# Доска домашних заданий

## Обзор

Система управления домашними заданиями для учебных заведений с разделением ролей пользователей.

## Модели данных

### User (расширенная модель пользователя)
- Роли: студент, преподаватель, администратор
- Дополнительные поля: student_id, phone, birth_date

### Subject (предмет)
- Название, описание

### Assignment (домашнее задание)
- Предмет, преподаватель, заголовок, описание, срок сдачи, штрафы, максимальный балл

### Submission (сдача задания)
- Задание, студент, текст сдачи, дата сдачи, опоздание

### Grade (оценка)
- Сдача, баллы, комментарий, преподаватель, дата оценки

## Представления

- CRUD для заданий
- Сдача заданий студентами
- Оценивание преподавателями
- Админка для управления

## Формы

- Форма создания задания
- Форма сдачи задания
- Форма оценки

## URL маршруты

- `/assignments/` - список заданий
- `/assignments/create/` - создание задания
- `/assignments/<id>/` - детализация задания
- `/assignments/<id>/submit/` - сдача задания
- `/assignments/<id>/grade/` - оценка задания

## Шаблоны

Используются Bootstrap 5 для адаптивного дизайна.

## Настройка

1. Установить зависимости: `pip install django`
2. Выполнить миграции: `python manage.py migrate`
3. Создать суперпользователя: `python manage.py createsuperuser`
4. Запустить сервер: `python manage.py runserver`</content>
