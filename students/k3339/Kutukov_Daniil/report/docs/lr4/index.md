# Лабораторная работа 4: Frontend разработка с Vue.js

**Выполнил:** Кутуков Даниил

## Обзор

Лабораторная работа посвящена разработке современного веб-приложения с использованием Vue.js 3, TypeScript и Vuetify для создания интерфейса системы управления колледжем.

## Технологии

- **Vue.js 3** - Прогрессивный JavaScript фреймворк
- **TypeScript** - Типизированный JavaScript
- **Vuetify 3** - Material Design компоненты для Vue
- **Vue Router 4** - Официальный роутер для Vue.js
- **Pinia** - Современное состояние управления для Vue
- **Vite** - Быстрый инструмент сборки
- **Axios** - HTTP клиент для запросов к API

## Основные компоненты

### 1. Vue.js приложение (college-app)
Современное SPA с использованием Composition API и TypeScript.

### 2. Vuetify UI фреймворк
Material Design компоненты для создания красивого и отзывчивого интерфейса.

### 3. Vue Router
Система маршрутизации для навигации между страницами приложения.

### 4. Pinia Store
Централизованное управление состоянием приложения.
