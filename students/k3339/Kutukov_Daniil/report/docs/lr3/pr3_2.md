## API для системы воинов

Разработка REST API для управления воинами, их профессиями и навыками с использованием Django REST Framework.

## Модели данных

### Warrior (Воин)
```python
class Warrior(models.Model):
    name = models.CharField(max_length=100)
    race = models.CharField(max_length=50)
    level = models.IntegerField(default=1)
```

### Profession (Профессия)
```python
class Profession(models.Model):
    title = models.CharField(max_length=100)
    description = models.TextField(blank=True)
```

### Skill (Навык)
```python
class Skill(models.Model):
    name = models.CharField(max_length=100)
    description = models.TextField(blank=True)
    warriors = models.ManyToManyField(Warrior, related_name='skills')
```

## API Endpoints

### APIView endpoints
- `GET /war/api/warriors/` - Список всех воинов
- `POST /war/api/warriors/` - Создание нового воина
- `GET /war/api/warriors/{id}/` - Детали воина
- `DELETE /war/api/warriors/{id}/` - Удаление воина

### GenericView endpoints
- `GET /war/api/professions/` - Список профессий
- `POST /war/api/professions/` - Создание профессии
- `GET /war/api/professions/{id}/` - Детали профессии

### ViewSet endpoints
- `GET /war/api/skills/` - Список навыков
- `POST /war/api/skills/` - Создание навыка
- `GET /war/api/skills/{id}/` - Детали навыка
- `PUT /war/api/skills/{id}/` - Обновление навыка
- `DELETE /war/api/skills/{id}/` - Удаление навыка

## Тестирование API

### Примеры запросов

```python
# Получение списка воинов
GET /war/api/warriors/

# Создание нового воина
POST /war/api/warriors/
{
    "name": "Артур",
    "race": "Человек",
    "level": 10
}

# Получение списка профессий
GET /war/api/professions/

# Создание навыка
POST /war/api/skills/
{
    "name": "Меч",
    "description": "Навык владения мечом"
}
```

## Архитектура представлений

### APIView
Используется для простых операций CRUD с дополнительной логикой.

### Generic Views
Предоставляют готовые реализации для стандартных операций.

### ViewSets
Объединяют несколько представлений в один класс с автоматической маршрутизацией.
