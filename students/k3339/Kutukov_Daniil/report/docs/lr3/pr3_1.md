## Задание 1: Создание объектов

Скрипт для создания тестовых данных в системе управления автовладельцами.

### Код создания объектов

```python
# Создание автовладельцев
owner1 = Owner.objects.create(
    last_name="Иванов", first_name="Олег", birth_date=datetime(1985, 3, 15)
)

# Создание автомобилей
car1 = Car.objects.create(
    license_plate="A123BC", brand="Toyota", model="Camry", color="Белый"
)

# Создание водительских удостоверений
license1 = DriverLicense.objects.create(
    owner=owner1,
    license_number="1234567890",
    type="B",
    issue_date=datetime(2005, 3, 15)
)

# Создание связей владения
ownership1 = Ownership.objects.create(
    owner=owner1, car=car1, start_date=datetime(2010, 5, 10)
)
```

## Задание 2: Фильтрация запросов

Примеры запросов к базе данных с использованием Django ORM.

### Фильтрация по марке автомобиля
```python
# Все автомобили марки Toyota
toyota_cars = Car.objects.filter(brand="Toyota")
```

### Фильтрация по году выпуска
```python
# Автомобили старше 2010 года
old_cars = Car.objects.filter(year__lt=2010)
```

### Сложные запросы
```python
# Владельцы с несколькими автомобилями
owners_with_multiple_cars = Owner.objects.annotate(
    car_count=Count('ownership')
).filter(car_count__gt=1)
```

## Задание 3: Агрегация запросов

Примеры агрегационных запросов.

### Подсчет количества автомобилей по маркам
```python
from django.db.models import Count

car_counts = Car.objects.values('brand').annotate(
    count=Count('id')
).order_by('-count')
```

### Средний возраст владельцев
```python
from django.db.models import Avg
from datetime import date

average_age = Owner.objects.aggregate(
    avg_age=Avg(date.today().year - models.F('birth_date__year'))
)
```

### Статистика по типам удостоверений
```python
license_stats = DriverLicense.objects.values('type').annotate(
    count=Count('id')
)
```
