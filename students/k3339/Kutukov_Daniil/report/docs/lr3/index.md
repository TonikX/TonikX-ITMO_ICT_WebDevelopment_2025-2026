# Лабораторная работа 3: Продвинутые Django приложения

**Выполнил:** Кутуков Даниил

## Обзор

Лабораторная работа посвящена разработке комплексной системы управления колледжем с использованием Django REST Framework. Проект включает модели данных для студентов, преподавателей, курсов и оценок, а также REST API для взаимодействия с данными.
