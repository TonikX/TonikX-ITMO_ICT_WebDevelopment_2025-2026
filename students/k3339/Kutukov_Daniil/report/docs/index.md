## Лабораторные работы

### Лабораторная работа 1: Основы сетевого программирования
Изучение клиент-серверной архитектуры на Python: UDP эхо-сервер, TCP сервер для математических операций, HTTP серверы, многопоточный чат и веб-форма для оценок.

### Лабораторная работа 2: Django веб-приложения
Разработка двух Django проектов: системы управления домашними заданиями с ролевой моделью и системы учета автовладельцев с моделями автомобилей и водительских удостоверений.

### Лабораторная работа 3: Продвинутые Django приложения
Комплексная система управления колледжем с Django REST Framework: модели для студентов, преподавателей, дисциплин, групп и оценок; REST API; практические работы по созданию объектов и API тестированию.

### Лабораторная работа 4: Frontend разработка с Vue.js
Современное SPA на Vue.js 3 с TypeScript, Vuetify, Pinia и Vue Router для интерфейса системы управления колледжем.

## Технологии

- **Backend:** Python, Django 4-5, Django REST Framework, SQLite
- **Frontend:** HTML, CSS, JavaScript, Vue.js 3, TypeScript, Vuetify 3, Pinia, Vue Router
- **Инструменты:** Git, MkDocs, GitHub Pages, Socket programming, HTTP protocols
