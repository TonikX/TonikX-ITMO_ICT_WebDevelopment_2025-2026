# Лабораторная работа 1: Работа с сокетами

**Выполнил:** Кутуков Даниил

## Обзор

Лабораторная работа посвящена изучению основ сетевого программирования с использованием сокетов в Python. Реализованы различные типы клиент-серверных приложений: UDP эхо-сервер, TCP сервер для математических вычислений, простой HTTP сервер, многопоточный чат-сервер и HTTP сервер с веб-формой.

## Технологии

- Python 3
- Socket модуль
- Threading для многопоточности
- Tkinter для GUI
