# Дисциплины

**Базовый URL:** `/api/subjects/`  
**Требуется аутентификация:** Да

## Модель

| Поле | Тип | Описание |
|------|-----|----------|
| id | integer | ID дисциплины |
| name | string | Название дисциплины (уникальное) |
| total_hours | integer | Общее количество часов |

## Эндпоинты

### Список дисциплин
`GET /api/subjects/`

### Создать дисциплину
`POST /api/subjects/`

```json
{
  "name": "Математический анализ",
  "total_hours": 144
}
```

### Получить дисциплину
`GET /api/subjects/{id}/`

### Обновить дисциплину
`PUT/PATCH /api/subjects/{id}/`

### Удалить дисциплину
`DELETE /api/subjects/{id}/`
