# Оценки

**Базовый URL:** `/api/grades/`  
**Требуется аутентификация:** Да

## Модель

| Поле | Тип | Описание |
|------|-----|----------|
| id | integer | ID оценки |
| student | integer | ID студента |
| subject | integer | ID дисциплины |
| grade | integer | Оценка (2-5) |
| semester | integer | Семестр (1-8) |
| date | date | Дата выставления |

## Эндпоинты

### Список оценок
`GET /api/grades/`

Возвращает детальную информацию с вложенными объектами студента и дисциплины.

### Создать оценку
`POST /api/grades/`

```json
{
  "student": 1,
  "subject": 1,
  "grade": 5,
  "semester": 1,
  "date": "2024-12-15"
}
```

### Получить оценку
`GET /api/grades/{id}/`

### Обновить оценку
`PUT/PATCH /api/grades/{id}/`

### Удалить оценку
`DELETE /api/grades/{id}/`

## Ограничения

- Оценка должна быть от 2 до 5
- Семестр должен быть от 1 до 8
