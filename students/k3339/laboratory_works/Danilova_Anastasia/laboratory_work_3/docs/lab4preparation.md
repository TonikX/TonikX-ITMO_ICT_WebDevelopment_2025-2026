## Подготовка к работе

### Установка библиотек

Изначально был установлен node.js и npm и создан проект Vue.js Parks.
А также установлены библиотеки:

* axios - Для HTTP запросов
* vue-router - Для  маршрутизации
* pinia - Для хранения состояния (например токена)
* vuetify - Для стилей
* @mdi/font - для UI компонентов


Также был настроен CORS, о нем расписано **[здесь](cors.md)**.

### Структура проекта

```text

├── src/
│   ├── assets/          # Статические файлы
│   ├── components/      # Компоненты
│   ├── views/          # Страницы
│   ├── router/         # Маршрутизация
│   ├── store/          # Хранилище Pinia
│   ├── services/       # API сервисы
│   └── App.vue         # Главный компонент
```
