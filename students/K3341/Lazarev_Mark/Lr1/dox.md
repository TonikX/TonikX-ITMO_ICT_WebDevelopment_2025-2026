## 1. UDP Клиент-Сервер

Описание:  
Приложение состоит из клиента и сервера, которые обмениваются сообщениями по UDP. Клиент отправляет текст серверу, сервер отображает адрес отправителя и сообщение, а затем отправляет ответ "Hello, client".

Проверка работы:  
Сервер:  
python3 udp_server.py  

Клиент:  
python3 udp_client.py  

Пример:  
- Клиент вводит: Hello, server  
- Сервер выводит адрес клиента и текст сообщения  
- Клиент получает: Hello, client

## 2. TCP Клиент-Сервер (вычисление гипотенузы)

Описание:  
TCP сервер принимает два числа от клиента — катеты прямоугольного треугольника, вычисляет гипотенузу и возвращает результат.

Проверка работы:  
Сервер:  
python3 server_pythagoras.py  

Клиент:  
python3 client.py  

Пример:  
- Ввести два числа: 3 4  
- Сервер возвращает: Гипотенуза для катетов 3.0 и 4.0: 5.000

## 4. Многопользовательский чат (TCP + потоки)

Описание:  
Сервер позволяет подключать несколько клиентов одновременно. Все сообщения от одного клиента пересылаются остальным. Потоки обеспечивают параллельную обработку соединений.

Проверка работы:  
Сервер:  
python3 chat_server.py  

Клиенты (отдельные окна):  
python3 chat_client.py  

Пример:  
- Вводить сообщения — они будут видны всем участникам  
- Для выхода использовать команду: /quit

## 5. TCP Журнал оценок (HTML + GET/POST)

Описание:  
Сервер хранит список оценок по предметам и формирует журнал в виде HTML-страницы.  
- GET — отображение текущего журнала  
- POST — добавление новой оценки

Проверка работы:  
Сервер:  
python3 journal_server.py  

Просмотр журнала в браузере:  
http://localhost:8082  

Просмотр журнала через терминал:  
curl http://localhost:8082  

Добавление оценки (POST-запрос):  
curl -X POST -d "subj=Математика&grade=5" http://localhost:8082  

После отправки новая оценка автоматически появляется в журнале на странице.
