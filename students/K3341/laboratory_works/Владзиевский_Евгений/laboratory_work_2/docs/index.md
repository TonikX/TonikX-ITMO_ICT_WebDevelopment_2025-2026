# Отчёт — Conference Service

Краткое описание проекта: веб-сервис для управления конференциями с регистрацией участников, отзывами и административным интерфейсом.

В отчёте описаны установка, запуск, веб-интерфейс, API, администрирование и скриншоты интерфейса.

Содержание доступно в навигации слева.
