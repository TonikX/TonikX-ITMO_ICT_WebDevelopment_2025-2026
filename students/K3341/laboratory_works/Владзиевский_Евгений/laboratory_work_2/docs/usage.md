# Запуск и использование

Запуск сервиса в режиме разработки:

```bash
uvicorn main:app --reload --port 8001
```

Откройте веб-интерфейс: http://127.0.0.1:8001/ui/

Рекомендуемые проверки:
- Регистрация пользователя
- Вход
- Просмотр списка конференций
- Регистрация на выступление
- Добавление отзыва
- Просмотр таблицы участников
