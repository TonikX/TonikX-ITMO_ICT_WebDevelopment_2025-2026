# Документация лабораторных работ

## Возможности

- **Аутентификация**: регистрация, вход, обновление JWT токенов.
- **Посты**: создание, получение и управление публикациями.
- **Взаимодействия**: лайки и комментарии.
- **Медиа**: работа с изображениями (заготовки для интеграции).
