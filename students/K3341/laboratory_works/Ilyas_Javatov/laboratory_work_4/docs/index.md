# School System API

Серверная часть системы для завуча школы. Реализована на Django, Django REST Framework
и Djoser с использованием PostgreSQL (или любой СУБД через Django ORM).

Возможности:
- управление учителями, классами, кабинетами и предметами;
- назначения учителей и расписание занятий;
- учет учеников и четвертных оценок;
- отчеты и аналитические запросы.

Документация:
- **Модели данных** — сущности и связи;
- **API эндпоинты** — доступные запросы и параметры.
