    Документация Frontend интерфейсов

    Добро пожаловать в документацию клиентских интерфейсов системы управления отелем, разработанных с использованием Vue 3 и Vuetify.

    Описание

    Данная система предоставляет пользовательские интерфейсы для работы с системой управления отелем, включая авторизацию, регистрацию и управление профилем пользователя.

    Технологический стек

    • Vue 3 - Прогрессивный JavaScript фреймворк для создания пользовательских интерфейсов
    • Vuetify 3 - Material Design компонентная библиотека для Vue
    • Vue Router - Официальный маршрутизатор для Vue.js
    • Axios - HTTP клиент для взаимодействия с API
    • Vite - Современный инструмент сборки для frontend проектов

    Основные возможности

    • Авторизация пользователей
    • Регистрация новых пользователей
    • Управление профилем пользователя
    • Изменение пароля
    • Защита маршрутов на основе аутентификации
    • Управление типами номеров
    • Управление этажами
    • Управление номерами
    • Управление гостями
    • Управление проживаниями
    • Управление сотрудниками
    • Управление графиком уборки

    Быстрый старт

    Установка зависимостей

    npm install

    Запуск development сервера

    npm run dev

    Приложение будет доступно по адресу http://localhost:5173

    Сборка для production

    npm run build

    Просмотр production сборки

    npm run preview

    Взаимодействие с Backend

    Frontend приложение взаимодействует с Django REST Framework API, развернутым на http://localhost:8000.

    Требования к Backend

    • Django сервер должен быть запущен на порту 8000
    • CORS должен быть настроен для разрешения запросов с frontend
    • API endpoints должны быть доступны по стандартным путям Djoser
