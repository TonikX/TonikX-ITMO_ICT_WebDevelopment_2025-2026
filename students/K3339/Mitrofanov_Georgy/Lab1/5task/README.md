# Задание 5 — простой веб-сервер (GET/POST) + журнал оценок

Сервер:
- принимает POST с дисциплиной и оценкой
- хранит оценки **структурировано**: у каждой дисциплины список оценок
- по GET отдаёт HTML-страницу со всеми дисциплинами и их оценками

## Запуск
```bash
python server.py
```

Открыть в браузере:
- http://127.0.0.1:8081/

## Как добавить оценку (варианты)
1) Через форму на главной странице (самый простой)
2) Через curl:
```bash
curl -X POST -d "subject=Математика&grade=5" http://127.0.0.1:8081/
```
