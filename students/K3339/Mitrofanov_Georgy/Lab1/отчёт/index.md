# Отчёт по лабораторной работе №1: «Web‑сокеты / сетевые приложения на Python»

## Цель работы
Освоить практические навыки сетевого программирования на Python с использованием библиотеки `socket`: работа по UDP и TCP, создание простого HTTP‑сервера, многопользовательский чат, а также обработка HTTP GET/POST.

## Используемые технологии
- Python 3.6+  
- Библиотека `socket` (UDP/TCP)
- `threading` (для многопользовательского чата)
- Простые HTTP‑ответы (формирование заголовков/тела ответа)
- Хранение данных в JSON (для «журнала оценок» в задании 5)

## Постановка задачи
В рамках лабораторной работы выполнены 5 заданий:

1. **UDP “Hello”**: клиент отправляет серверу `Hello, server`, сервер отвечает `Hello, client`.
2. **TCP мат. операции**: клиент вводит параметры, сервер вычисляет результат и возвращает клиенту.  
   **Вариант 4** — площадь параллелограмма (`S = a * h`).
3. **HTTP + файл**: сервер отдаёт HTML‑страницу, загружая её из файла `index.html`.
4. **Многопользовательский чат**: TCP‑чат с поддержкой нескольких клиентов (каждый клиент запускается отдельным `client.py`), обработка подключений в потоках.
5. **HTTP GET/POST + журнал**: мини‑вебсервер, который принимает оценку и предмет, хранит оценки структурировано (у предмета список оценок) и выдаёт HTML со всем журналом.

## Итог
Реализованы отдельные клиент/сервер приложения для UDP и TCP, простой HTTP‑сервер, многопользовательский чат и веб‑сервер с обработкой POST‑формы и хранением оценок по дисциплинам.
