# Интерфейсы и способы взаимодействия

Ниже перечислено, как именно взаимодействовать с каждым заданием (терминал/браузер). Скриншоты можно вставить после соответствующих пунктов.

## Задание 1 — UDP “Hello”
### Сервер
- Запуск: `python server.py`
- В консоли сервера должно появиться принятое сообщение `Hello, server`

### Клиент
- Запуск: `python client.py`
- В консоли клиента должно появиться сообщение от сервера `Hello, client`

> Скриншоты: терминал сервера и терминал клиента.

---

## Задание 2 — TCP мат. операции (вариант 4)
### Сервер
- Запуск: `python server.py`
- Сервер ожидает подключение, затем получает JSON‑запрос и отправляет JSON‑ответ

### Клиент
- Запуск: `python client.py`
- Клиент попросит ввести:
  - сторону `a`
  - высоту `h`
- После ввода выводит результат площади параллелограмма

> Скриншоты: ввод параметров и вывод результата.

---

## Задание 3 — HTTP сервер, отдающий файл `index.html`
### Сервер
- Запуск: `python server.py`
- Сервер слушает порт `8080`

### Браузер
- Открыть страницу:
  - http://127.0.0.1:8080/
- Должна отобразиться HTML‑страница из файла `index.html`

> Скриншоты: страница в браузере + терминал сервера (по желанию).

---

## Задание 4 — многопользовательский чат (TCP + threading)
### Сервер
- Запуск: `python server.py`
- Сервер слушает порт чата и принимает подключения

### Клиенты
- Каждый клиент — отдельный терминал:
  ```bash
  python client.py
  ```
- Можно запустить 3+ клиента и писать сообщения друг другу
- Команда:
  - `/exit` — выйти

> Скриншоты: несколько окон терминала с перепиской.

---

## Задание 5 — мини‑вебсервер + журнал оценок (GET/POST)
### Сервер
- Запуск: `python server.py`
- Сервер слушает `8081`
- Данные сохраняются в `journal.json`

### Браузер (форма)
- Открыть:
  - http://127.0.0.1:8081/
- Ввести:
  - дисциплину
  - оценку
- Нажать «Добавить» → на странице появится обновлённый журнал

### Через curl
```bash
curl -X POST -d "subject=Математика&grade=5" http://127.0.0.1:8081/
```

> Скриншоты: страница журнала + добавление оценки.
