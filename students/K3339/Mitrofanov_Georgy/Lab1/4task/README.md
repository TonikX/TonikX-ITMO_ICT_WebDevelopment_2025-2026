# Задание 4 — многопользовательский чат (TCP + threading)

Требование: чат должен быть многопользовательским, запуск каждого клиента — одним файлом `client.py`.
То есть можно запустить 3 терминала с `python client.py` и общаться втроём.

## Запуск
1) Сервер:
```bash
python server.py
```

2) Клиенты (в разных терминалах, сколько нужно):
```bash
python client.py
```

Команды в чате:
- `/exit` — выйти
