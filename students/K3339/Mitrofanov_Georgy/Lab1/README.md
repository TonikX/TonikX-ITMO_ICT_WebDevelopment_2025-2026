# Web-программирование

## Структура лабораторных работ

1task — UDP "Hello, server" / "Hello, client"  
2task — TCP сервер вычислений (несколько операций)  
3task — HTTP-ответ с HTML из файла `index.html`  
4task — Многопользовательский TCP-чат (threading)  
5task — Мини-вебсервер (GET/POST) + журнал оценок
