# Доска домашних заданий (Django 3 + PostgreSQL)

Проект реализует:
- Регистрацию пользователей (ученики).
- Просмотр домашних заданий по всем дисциплинам (сроки, описание).
- Сдачу домашнего задания в текстовом виде (ученик сдаёт/редактирует свою сдачу).
- Оценивание учителем через Django-admin (оценка ставится объекту Submission).
- Страницу-таблицу с оценками всех учеников класса по всем заданиям.

## Структура
- `homework_board/` — настройки Django-проекта
- `board/` — приложение с логикой "доски"
- `board/templates/board/` — HTML-шаблоны
- `board/templatetags/` — фильтр шаблонов для вывода оценок в таблице

## Требования
- Python 3.6+
- PostgreSQL
- Django 3.x

## Быстрый старт (локально)

1) Создайте виртуальное окружение и установите зависимости:
```bash
python -m venv venv
# Windows: venv\Scripts\activate
source venv/bin/activate

pip install -r requirements.txt
```

2) Создайте базу данных PostgreSQL (пример):
```sql
CREATE DATABASE LR2;
```

3) Настройте подключение к БД в `homework_board/settings.py`:
- NAME, USER, PASSWORD, HOST, PORT

4) Примените миграции:
```bash
python manage.py migrate
```

5) Создайте учителя (админа):
```bash
python manage.py createsuperuser
```

6) Запустите сервер:
```bash
python manage.py runserver
```

## Как пользоваться (взаимодействие)

### 1) Учитель (администратор)
(kirillkoshkarev - kirill567)
1. Зайдите в админку: `http://127.0.0.1:8000/admin/`
2. Добавьте домашние задания в разделе **Homeworks**:
   - subject (предмет)
   - teacher (преподаватель)
   - issued_date (дата выдачи)
   - start_date / due_date (период выполнения)
   - task_text (текст задания)
   - penalties (штрафы)
3. Когда ученики сдадут задания, появятся записи **Submissions**.
   Откройте Submission и выставьте:
   - grade (оценка 1–10)
   - teacher_comment (комментарий учителя, опционально)

### 2) Ученик (обычный пользователь)
1. Откройте главную страницу: `http://127.0.0.1:8000/`
2. Зарегистрируйтесь: кнопка **Регистрация**
3. Откройте нужное ДЗ и нажмите **Сдать / изменить сдачу**
4. Введите ответ текстом и сохраните
5. Оценка появится после того, как учитель выставит её в админке

### 3) Таблица оценок всего класса
- Откройте страницу: `http://127.0.0.1:8000/class-grades/`
- Там будет таблица: строки — ученики, колонки — задания, значения — оценки или `-`.

## Примечания
- "Ученики" в таблице — все пользователи, кроме `is_staff` и `is_superuser`.
- Одна сдача на одно ДЗ на одного ученика (ограничение `unique_together`).

## Полезные команды
Создать миграции после изменения моделей:
```bash
python manage.py makemigrations
python manage.py migrate
```
