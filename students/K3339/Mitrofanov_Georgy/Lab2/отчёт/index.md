# Отчёт по лабораторной работе: «Доска домашних заданий»

## Цель работы
Овладеть практическими навыками разработки web-приложения на Django с хранением данных в PostgreSQL, реализовать регистрацию пользователей, выдачу и сдачу домашних заданий, выставление оценок и отображение успеваемости.

## Используемые технологии
- Python 3.6+ (в проекте использовался Python 3.9)
- Django 3.x (в проекте: Django 3.2.x)
- PostgreSQL
- Django Templates (HTML-шаблоны)
- Bootstrap 5 (подключение через CDN) для базового оформления интерфейса
- MkDocs + mkdocs-material для оформления отчёта в виде сайта документации

## Постановка задачи
О домашнем задании должна храниться следующая информация:
- предмет
- преподаватель
- дата выдачи
- период выполнения (начало/дедлайн)
- текст задания
- информация о штрафах

Необходимо реализовать функционал:
1. Регистрация новых пользователей.
2. Просмотр домашних заданий по всем дисциплинам (сроки выполнения, описание задания).
3. Сдача домашних заданий в текстовом виде.
4. Администратор (учитель) должен иметь возможность поставить оценку за задание средствами Django-admin.
5. В клиентской части должна формироваться таблица, отображающая оценки всех учеников класса.
   - Учитель видит оценки всех учеников.
   - Ученик видит только свои оценки.

## Итог
Реализован сайт «Доска домашних заданий» на Django с PostgreSQL, содержащий:
- регистрацию и вход/выход пользователей;
- список домашних заданий и детальную страницу задания;
- сдачу задания в виде текста;
- выставление оценок через Django-admin;
- страницу «Оценки класса» для учителя;
- страницу «Мои оценки» для ученика;
- базовое оформление интерфейса с использованием Bootstrap 5.
