# Работа учителя (администратора)

## Вход в админку
Адрес: http://127.0.0.1:8000/admin/

## Создание домашнего задания
Раздел `Homework` → «Добавить» → заполнить поля → «Сохранить».

## Выставление оценки
Раздел `Submission` → выбрать сдачу ученика → поле `grade` → «Сохранить».

## Оценки класса
Учитель открывает страницу «Оценки класса» и видит общую таблицу оценок.
