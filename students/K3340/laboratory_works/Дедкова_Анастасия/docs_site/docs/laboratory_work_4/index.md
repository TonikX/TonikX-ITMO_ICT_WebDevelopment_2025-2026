# Лабораторная работа 4  
## Обзор

Цель работы - разработать клиентское приложение на **Vue 3**, которое взаимодействует с API библиотеки, созданным ранее.  
Приложение реализовано как SPA (Single Page Application) и включает авторизацию, работу с читателями, выдачами и экземплярами книг.

## Что реализовано

### Возможности обычного пользователя
- вход и регистрация;
- просмотр каталога книг;
- просмотр «Профиля читателя»;
- просмотр своих активных выдач;
- редактирование контактных данных (телефон, адрес).

### Возможности администратора
- работа с выдачами книг;
- просмотр всех активных выдач;
- управление экземплярами книг (добавление, списание);
- просмотр отчётов по читателям;
- переход в Django-admin.

## Используемые технологии
- Vue 3 (Composition API)
- Vue Router
- Pinia (хранилище пользователя и токена)
- Axios (взаимодействие с API)
- Vite (сборка проекта)

## Основные страницы
- `/books` - каталог книг  
- `/profile` - профиль читателя  
- `/on-loan` - книги на руках  
- `/manage-loans` - выдача книг  
- `/manage-copies` - экземпляры книг  
- `/reports` - отчёты  
- `/login` и `/register` - авторизация  

Навигация и доступ зависят от роли пользователя.
