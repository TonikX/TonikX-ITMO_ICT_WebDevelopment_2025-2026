# Практическая работа №3  
## Расширение пользовательской модели и вывод информации о владельцах

### Цель работы
Реализовать расширенную модель пользователя в Django, чтобы каждый пользователь мог быть владельцем автомобиля.

Новые поля пользователя:

- номер паспорта;
- домашний адрес;
- национальность.

---

## Выполненные шаги

### 1. Создано приложение `accounts`
Добавлена кастомная модель пользователя.

Файл:  
`accounts/models.py`

### 2. Реализована расширенная модель пользователя

Добавлены поля:

- `passport_number`
- `home_address`
- `nationality`

Модель наследует `AbstractUser`.

### 3. Указана кастомная модель пользователя

В файле настроек:

```python
AUTH_USER_MODEL = 'accounts.User'
```

### 4. Настроена админ-панель

В разделе Users отображаются новые поля пользователя.
Файл:
`accounts/admin.py`

### 5. Реализована форма создания пользователя

Страница `/owners/create/` позволяет создать нового владельца с заполнением всех атрибутов.
Файл:
`accounts/forms.py`

### 6. Выведена таблица владельцев

На странице `/owners/` выводится таблица пользователей:

- логин

- имя

- фамилия

- паспорт

- адрес

- национальность

Файл шаблона:
`templates/owners_list.html`
