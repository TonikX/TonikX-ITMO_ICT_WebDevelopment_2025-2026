# Лабораторная работа 2 по предмету Web-программирование

---

## Описание

Реализовано **веб-приложение на Django**, выполняющее функции **онлайн-табло авиарейсов** с возможностью бронирования билетов и написания отзывов.
Приложение разделено на пользовательскую и административную части.

**Функциональные возможности:**

* Просмотр всех рейсов + фильтры.
* Просмотр детальной информации о рейсе.
* Бронирование мест пользователем.
* Просмотр и удаление своих бронирований.
* Добавление отзывов к рейсу.
* Удаление своих отзывов.
* Админ может просматривать пассажиров рейса, добавлять и удалять брони.

---

## Требования

* Python **3.9+**
* Django **3.2 LTS**
* Библиотеки:

  * `django-filter`
  * `Bootstrap 5` (через CDN)

## Как работает приложение

1. **Гость** может просматривать рейсы и карточки рейсов.
2. **Авторизованный пользователь** может:

   * забронировать билет;
   * просматривать свои брони;
   * удалять свои брони;
   * оставлять отзывы;
   * удалять свои отзывы.
3. **Администратор** может:

   * просматривать список пассажиров выбранного рейса;
   * добавлять бронь на пользователя вручную;
   * удалять любую бронь.

## Примеры работы

### Главная страница

Отображает список рейсов с базовой информацией: авиакомпания, направление, дата и количество свободных мест.

### Детали рейса

Показывает полную информацию о рейсе:

* код авиакомпании,
* время вылета и прибытия,
* количество мест,
* отзывы пользователей.

Если пользователь авторизован, появляются кнопки:

* «Забронировать»;
* «Оставить отзыв»;
* «Удалить свой отзыв».

---

## Пример работы с отзывами

### Добавление отзыва

1. Нажать **«Оставить отзыв»**.
2. Заполнить поля: дата рейса, текст, рейтинг (1–10).
3. Отправить форму — отзыв появится внизу карточки рейса.

### Удаление отзыва

* У кнопки **«Удалить»** рядом с отзывом появляется форма.
* Отображается только у **автора** или **админа**.
* После удаления страница перезагружается с обновлённым списком.

---

## Пример работы с бронированием

### Пользователь

* Может забронировать место на странице рейса.
* В разделе **«Мои бронирования»** отображаются все активные брони.
* Удаление брони возвращает место в доступные.

### Администратор

* На странице **«Пассажиры рейса»** видит всех пассажиров и их билеты.
* Может:

  * добавить бронь вручную;
  * удалить бронь любого пользователя.

## Основные URL-маршруты

| Маршрут                         | Назначение                 |
| ------------------------------- | -------------------------- |
| `/`                             | Список рейсов              |
| `/flights/<id>/`                | Подробности рейса          |
| `/flights/<id>/book/`           | Бронирование рейса         |
| `/my-bookings/`                 | Мои брони                  |
| `/reviews/<id>/create/`         | Создать отзыв              |
| `/reviews/<id>/delete/`         | Удалить отзыв              |
| `/flights/<id>/passengers/`     | Пассажиры рейса (админ)    |
| `/flights/<id>/passengers/add/` | Добавить пассажира (админ) |

## Модели данных

### Flight

| Поле            | Тип          | Описание       |
| --------------- | ------------ | -------------- |
| airline         | FK           | Авиакомпания   |
| number          | CharField    | Номер рейса    |
| direction       | Choice       | Вылет/Прибытие |
| seats_total     | IntegerField | Всего мест     |
| seats_available | IntegerField | Доступно мест  |

### Booking

| Поле          | Тип       | Описание                   |
| ------------- | --------- | -------------------------- |
| user          | FK        | Пользователь               |
| flight        | FK        | Рейс                       |
| ticket_number | CharField | Номер билета               |
| seats_count   | Integer   | Количество мест 		 |

### Review

| Поле        | Тип       | Описание      |
| ----------- | --------- | ------------- |
| author      | FK        | Пользователь  |
| flight      | FK        | Рейс          |
| body        | TextField | Текст отзыва  |
| rating      | Integer   | Оценка 1–10   |
| flight_date | Date      | Дата перелёта |
