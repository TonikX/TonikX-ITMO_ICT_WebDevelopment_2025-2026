# Лабораторная работа №3 — Library API

В этой работе реализована серверная часть библиотечной системы:

- Django ORM модели
- CRUD API (DRF ViewSets)
- Авторизация по JWT (Djoser + SimpleJWT)
- Фильтрация, поиск, кастомные запросы
- Документация (MkDocs)
