# Лабораторная работа 1

В рамках лабы разобрался с работой сокетов в Python. Написал клиент-серверные приложения на UDP и TCP, поднял простой HTTP сервер, который отдает HTML из файла, сделал многопользовательский чат с потоками и веб-сервер с обработкой GET/POST запросов. В целом стало понятно как работает сетевое взаимодействие на низком уровне и как формируются HTTP ответы руками без фреймворков
