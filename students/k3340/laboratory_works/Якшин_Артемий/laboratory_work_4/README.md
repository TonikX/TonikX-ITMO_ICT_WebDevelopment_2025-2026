# Лабораторная работа №4

## Реализация клиентской части приложения средствами Vue.js

### Описание задания

Реализация клиентского приложения для системы управления издательской деятельностью.

### Выполненные задачи

1. **Настройка CORS** для серверной части (лабораторная работа №3)
   - Добавлен пакет `django-cors-headers`
   - Настроены разрешённые источники для Vue.js приложения

2. **Реализация авторизации и регистрации**
   - Страница входа в систему
   - Страница регистрации
   - Управление профилем пользователя
   - Смена пароля

3. **Реализация клиентских интерфейсов**
   - Сотрудники (менеджеры, редакторы)
   - Авторы
   - Книги (с назначением авторов и редакторов)
   - Контракты на издание
   - Заказчики
   - Заказы (с позициями)

4. **Интерфейсы отчётов**
   - Книги по авторам
   - Ответственные редакторы
   - Количество редакторов на книгу
   - Контракты по месяцам
   - Топ менеджеры
   - Квартальный отчёт

5. **Подключение Vuetify**
   - Использована библиотека Vuetify 3
   - Настроена кастомная тема
   - Адаптивный дизайн

### Технологии

- Vue.js 3 (Composition API)
- Vuetify 3
- Vue Router 4
- Pinia
- Axios
- Vite

### Запуск

#### Backend (из laboratory_work_3)

```bash
cd ../laboratory_work_3/publishing_house
source venv/bin/activate
pip install django-cors-headers
python manage.py runserver
```

#### Frontend

```bash
cd publishing_frontend
npm install
npm run dev
```
