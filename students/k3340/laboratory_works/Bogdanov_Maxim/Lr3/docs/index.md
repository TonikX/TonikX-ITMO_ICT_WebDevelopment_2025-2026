## О проекте

School Management Service - это backend-сервис, разработанный на Go с использованием принципов Clean Architecture. Сервис предоставляет полный набор API для управления всеми аспектами школьной жизни: от учителей и учеников до расписания и оценок.

## Ключевые возможности

- **Управление пользователями**: Регистрация, аутентификация через JWT, ролевая модель доступа
- **Учителя**: CRUD операции, привязка к предметам и классам
- **Ученики**: Управление учениками, привязка к классам
- **Классы**: Управление классами и академическими годами
- **Расписание**: Гибкое управление расписанием занятий
- **Оценки**: Выставление и отслеживание оценок по периодам
- **Отчеты**: Аналитика успеваемости классов и учеников
- **Информационные запросы**: Статистика по учителям, ученикам, кабинетам

## Технологический стек

- **Язык**: Go 1.25+
- **Веб-фреймворк**: Chi Router
- **База данных**: PostgreSQL 18
- **Аутентификация**: JWT (Access & Refresh tokens)
- **Документация**: Swagger/OpenAPI 3.0
- **Контейнеризация**: Docker & Docker Compose
- **Миграции**: golang-migrate
- **Тестирование**: Go testing, testify

## Быстрый старт

Запустите проект за 2 минуты с помощью Docker:

```bash
# Клонируйте репозиторий
git clone <repo-url>
cd students/k3340/laboratory_works/Bogdanov_Maxim/Lr3

# Создайте .env файл (опционально)
echo "JWT_SECRET=your-secret-key" > .env

# Запустите все сервисы
docker compose up -d

# Проверьте статус
curl http://localhost:8080/health/live
```

Swagger UI будет доступен по адресу: [http://localhost:8080/swagger/](http://localhost:8080/swagger/)
