# Инструкция по запуску

## Требования

- Python 3.13
- pip
- SQLite3 (встроен в Python)

## Установка

### 1. Клонирование репозитория
```bash
git clone <URL_репозитория>
cd hotel_booking
```

### 2. Создание виртуального окружения

**Linux/macOS:**
```bash
python3 -m venv .venv
source .venv/bin/activate
```

**Windows:**
```cmd
python -m venv .venv
.venv\Scripts\activate
```

### 3. Установка зависимостей
```bash
pip install -r requirements.txt
```

### 4. Применение миграций
```bash
python manage.py migrate
```

### 5. Создание суперпользователя
```bash
python manage.py createsuperuser
```

### 6. Загрузка тестовых данных (опционально)
```bash
python manage.py seed_demo_data
```

При запросе о очистке существующих данных введите `y` или `n`.

**Будет создано:**
- 11 пользователей (8 обычных + 3 владельца отелей)
- 12 отелей в разных городах
- 15 типов удобств
- ~400-600 номеров
- ~160 бронирований (прошлые, текущие, будущие)
- ~100 отзывов

**Тестовые учётные данные:**
- Пользователи: `ivan_petrov`, `maria_sidorova`, и др. / пароль: `demo1234`
- Владельцы: `hotel_owner1`, `hotel_owner2`, `hotel_owner3` / пароль: `owner1234`

### 7. Запуск сервера разработки
```bash
python manage.py runserver
```

Сервер запустится на http://127.0.0.1:8000/

### 8. Запуск тестов
```bash
python manage.py test booking
```

---

## Первый запуск

После запуска сервера откройте браузер и перейдите на:

**Главная страница:** http://127.0.0.1:8000/

**Админ панель:** http://127.0.0.1:8000/admin/

---

## Структура URL

| URL | Описание |
|-----|----------|
| `/` | Список отелей |
| `/hotels/<id>/` | Детали отеля |
| `/hotels/<id>/rooms/` | Выбор номера в отеле |
| `/bookings/new/` | Создание бронирования |
| `/bookings/my/` | Мои бронирования |
| `/bookings/<id>/` | Детали бронирования |
| `/reviews/new/<booking_id>/` | Создание отзыва |
| `/reviews/my/` | Мои отзывы |
| `/login/` | Вход в систему |
| `/register/` | Регистрация |
| `/admin/` | Административная панель |
