# Лабораторная работа №2: Система бронирования отелей

## Цель работы

Разработать веб-приложение на Django для системы бронирования отелей с использованием SQLite3 в качестве базы данных.

## Основные задачи

1. Спроектировать и реализовать базу данных для системы бронирования
2. Реализовать систему аутентификации и авторизации пользователей
3. Создать функционал бронирования номеров с проверкой доступности
4. Разработать систему отзывов с валидацией прав доступа
5. Реализовать административную панель с возможностью управления бронированиями
6. Покрыть критичные компоненты тестами

## Технологический стек

- **Python:** 3.13
- **Django:** 5.2.7
- **База данных:** SQLite3
- **Frontend:** Django Templates + Bootstrap 5
- **Тестирование:** Django Test Framework

## Требования к функционалу

### Аутентификация
- Регистрация новых пользователей
- Вход/выход из системы
- Сброс пароля

### Бронирование
- Просмотр списка отелей с фильтрацией и поиском
- Просмотр доступных номеров отеля
- Создание бронирования с проверкой доступности
- Редактирование/удаление собственных бронирований (до заезда)
- Валидация дат и пересечений бронирований

### Отзывы
- Оставление отзывов после заезда
- Рейтинг от 1 до 10
- Привязка отзывов к конкретным бронированиям
- Редактирование/удаление собственных отзывов

### Административная панель
- Управление отелями, номерами, типами номеров
- Заселение и выселение гостей
- Просмотр всех бронирований и отзывов

### Дополнительно
- Таблица постояльцев за последние 30 дней
- Отображение среднего рейтинга отеля
- Пагинация списков
- Фильтрация номеров по параметрам
