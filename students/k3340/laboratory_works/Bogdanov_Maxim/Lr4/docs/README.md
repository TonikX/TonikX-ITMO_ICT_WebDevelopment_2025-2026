# Документация MkDocs

Эта директория содержит документацию в формате MkDocs.

## Установка

```bash
pip install -r requirements.txt
```

## Запуск

```bash
mkdocs serve
```

Документация будет доступна по адресу `http://localhost:8000`

## Сборка

```bash
mkdocs build
```

Собранная документация будет находиться в директории `site/`
