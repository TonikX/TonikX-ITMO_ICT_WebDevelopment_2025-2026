# Аутентификация и авторизация

## Страница входа (Login)

Реализована страница входа с полями:
- Имя пользователя
- Пароль

При успешной аутентификации пользователь получает токены доступа и обновления, которые сохраняются в localStorage.

## Страница регистрации (Register)

Позволяет новым пользователям зарегистрироваться в системе. После регистрации пользователь автоматически входит в систему.

## Управление токенами

### Хранение токенов
Токены доступа и обновления хранятся в localStorage браузера.

### Автоматическое обновление
При получении 401 ошибки (истекший токен) автоматически выполняется запрос на обновление токена. После успешного обновления исходный запрос повторяется.

### Защита маршрутов
Все маршруты, кроме страниц входа и регистрации, защищены навигационными хуками. Неавторизованные пользователи автоматически перенаправляются на страницу входа.
