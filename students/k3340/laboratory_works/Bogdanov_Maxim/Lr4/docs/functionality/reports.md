# Статистика и отчеты

## Статистика

Раздел статистики предоставляет следующие данные:

### Количество учителей по предметам
Таблица, показывающая сколько учителей преподают каждый предмет.

### Количество учеников по полу и классам
Таблица с разбивкой учеников по классам и полу.

### Количество кабинетов по типам
Информация о количестве кабинетов базовых и профильных дисциплин.

## Отчеты

### Отчет об успеваемости класса

Позволяет получить детальную информацию об успеваемости выбранного класса.

#### Возможности
- Выбор класса из выпадающего списка
- Просмотр общей информации о классе:
  - Количество учеников
  - Общий средний балл
  - Классный руководитель
- Таблица успеваемости по предметам:
  - Средняя оценка по каждому предмету
  - Количество оценок

#### Особенности
- Удобный выбор класса вместо ввода ID
- Наглядное представление данных в виде карточек и таблиц
