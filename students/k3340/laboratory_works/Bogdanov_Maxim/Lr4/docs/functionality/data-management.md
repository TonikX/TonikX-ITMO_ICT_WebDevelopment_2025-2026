# Управление данными

## Управление учителями

### Возможности
- Просмотр списка всех учителей
- Добавление нового учителя
- Редактирование данных учителя
- Удаление учителя

### Особенности
- Отображение кабинета учителя вместо ID
- Автоматическое заполнение формы при редактировании

## Управление учениками

### Возможности
- Просмотр списка учеников с фильтрацией по классу
- Добавление нового ученика
- Редактирование данных ученика
- Удаление ученика

### Особенности
- Фильтрация по классу в верхней части страницы
- Навигация из раздела "Классы" с предустановленным фильтром
- Отображение класса ученика вместо ID

## Управление классами

### Возможности
- Просмотр списка классов
- Добавление нового класса
- Редактирование данных класса
- Удаление класса

### Особенности
- Клик на название класса открывает список учеников этого класса
- Отображение учебного года и классного руководителя

## Управление расписанием

### Возможности
- Просмотр расписания с фильтрацией по классу
- Добавление нового урока
- Редактирование урока
- Удаление урока

### Особенности
- Отображение расписания в виде отдельных таблиц для каждого дня недели
- Фильтрация по классу
- Отображение всех данных в читаемом формате (предмет, учитель, кабинет)

## Управление оценками

### Возможности
- Просмотр списка оценок
- Добавление новой оценки
- Редактирование оценки
- Удаление оценки

### Особенности
- Отображение информации об ученике, предмете и периоде оценивания
- Все данные отображаются в понятном формате вместо ID
