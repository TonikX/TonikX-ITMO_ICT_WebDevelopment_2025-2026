# Обзор функционала

Приложение предоставляет полный набор функций для управления школьной системой.

## Основные разделы

### Аутентификация
- Вход в систему
- Регистрация новых пользователей
- Автоматическое обновление токенов

### Управление данными
- Учителя
- Ученики
- Классы
- Расписание
- Оценки

### Аналитика
- Статистика по различным показателям
- Отчеты об успеваемости классов

Подробное описание каждого раздела доступно в соответствующих разделах документации.
