## Описание директорий

### `src/api/`
Содержит конфигурацию HTTP клиента (Axios) с настройкой базового URL, обработкой токенов и ошибок.

### `src/stores/`
Pinia stores для управления состоянием:
- `auth.js` - состояние аутентификации
- `reference.js` - справочные данные

### `src/router/`
Конфигурация маршрутизации приложения с защитой маршрутов.

### `src/views/`
Vue компоненты страниц приложения. Каждый компонент отвечает за отображение и управление определенным разделом приложения.

### `src/plugins/`
Плагины Vue, включая конфигурацию Vuetify.

### `docs/`
Документация проекта в формате MkDocs, разбитая по тематическим разделам.
