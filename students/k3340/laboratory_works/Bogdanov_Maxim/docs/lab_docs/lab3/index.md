## Краткое описание

В ходе выполнения лабораторной работы была разработана серверная часть веб-приложения для управления школьной системой. Приложение реализовано на языке Go с использованием принципов Clean Architecture и предоставляет REST API для работы со всеми аспектами школьной жизни.

## Основные возможности

- Аутентификация и авторизация через JWT
- Управление учителями, учениками и классами
- Управление расписанием занятий
- Управление оценками
- Статистика и аналитика
- Генерация отчетов об успеваемости

## Технологический стек

- Go 1.25+
- PostgreSQL 18
- Chi Router
- JWT (Access & Refresh tokens)
- Docker & Docker Compose
- Swagger/OpenAPI 3.0
