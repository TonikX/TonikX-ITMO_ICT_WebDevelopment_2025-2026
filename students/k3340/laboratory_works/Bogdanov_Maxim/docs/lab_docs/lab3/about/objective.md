# Цель работы

Разработать серверную часть веб-приложения для управления школьной системой с использованием языка Go и принципов Clean Architecture.

## Задачи

1. Спроектировать архитектуру приложения на основе Clean Architecture
2. Реализовать REST API для работы с данными школы
3. Настроить базу данных PostgreSQL с миграциями
4. Реализовать систему аутентификации и авторизации через JWT
5. Добавить функционал статистики и отчетов
6. Обеспечить документацию API через Swagger
7. Настроить контейнеризацию через Docker
