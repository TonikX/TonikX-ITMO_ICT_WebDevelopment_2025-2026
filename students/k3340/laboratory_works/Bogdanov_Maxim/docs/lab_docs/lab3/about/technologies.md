# Используемые технологии

## Основной стек

### Go 1.25+
Язык программирования для разработки серверной части. Выбран за высокую производительность, простоту и отличную поддержку конкурентности.

### PostgreSQL 18
Реляционная база данных для хранения всех данных приложения. Используется для надежного хранения структурированных данных.

### Chi Router
Легковесный HTTP роутер для Go. Обеспечивает гибкую маршрутизацию и middleware поддержку.

## Аутентификация

### JWT (JSON Web Tokens)
Используется для аутентификации и авторизации пользователей. Реализована система с access и refresh токенами для безопасности.

## Документация

### Swagger/OpenAPI 3.0
Автоматическая генерация документации API на основе аннотаций в коде.

## Контейнеризация

### Docker & Docker Compose
Используется для контейнеризации приложения и упрощения развертывания. Включает контейнеры для базы данных, миграций и самого приложения.

## Миграции

### golang-migrate
Инструмент для управления миграциями базы данных. Позволяет версионировать схему БД и применять изменения последовательно.

## Тестирование

### Go testing + testify
Встроенный фреймворк тестирования Go с библиотекой testify для более удобных assertions.
