# Используемые технологии

## Основной стек

### Vue.js 3
Фреймворк для создания пользовательского интерфейса. Используется Composition API для более гибкой организации кода.

### Vue Router
Библиотека для маршрутизации в одностраничном приложении. Обеспечивает навигацию между страницами и защиту маршрутов.

### Pinia
Современная библиотека для управления состоянием приложения. Используется для хранения данных аутентификации и справочной информации.

### Vuetify 3
Библиотека компонентов Material Design для Vue.js. Обеспечивает готовые компоненты для создания современного интерфейса.

### Axios
HTTP клиент для выполнения запросов к API. Настроен с автоматическим добавлением токенов авторизации и обработкой ошибок.

### Vite
Современный инструмент сборки и разработки. Обеспечивает быструю разработку с hot module replacement.

## Дополнительные библиотеки

- **@mdi/font** - иконки Material Design Icons
- **vite-plugin-vuetify** - плагин для интеграции Vuetify с Vite
