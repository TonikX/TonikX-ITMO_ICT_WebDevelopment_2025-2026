# Цель работы

Разработать клиентскую часть веб-приложения для управления школьной системой с использованием Vue.js 3 и Vuetify.

## Задачи

1. Создать пользовательский интерфейс для работы с данными школы
2. Реализовать аутентификацию и авторизацию пользователей
3. Обеспечить взаимодействие с REST API серверной части
4. Реализовать CRUD операции для всех сущностей
5. Добавить функционал статистики и отчетов
6. Обеспечить удобный и интуитивный интерфейс
