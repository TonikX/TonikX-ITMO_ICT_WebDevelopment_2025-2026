## Краткое описание

В ходе выполнения лабораторной работы была разработана полнофункциональная клиентская часть веб-приложения для управления школьной системой. Приложение использует современные технологии Vue.js 3 и Vuetify, обеспечивает удобный пользовательский интерфейс и корректное взаимодействие с серверной частью через REST API.

## Основные возможности

- Аутентификация и авторизация пользователей
- Управление учителями, учениками и классами
- Управление расписанием занятий
- Управление оценками
- Просмотр статистики
- Генерация отчетов об успеваемости
