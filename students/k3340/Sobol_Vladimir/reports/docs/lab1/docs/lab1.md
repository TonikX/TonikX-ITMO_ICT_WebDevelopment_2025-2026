## Состав заданий
1. **UDP-клиент и сервер** — обмен приветственными сообщениями.  
2. **TCP-клиент и сервер** — выполнение математической операции (теорема Пифагора).  
3. **HTTP-сервер** — отдача страницы `index.html`.  
4. **Многопользовательский чат** (TCP + threading).  
5. **Веб-сервер для приёма и отображения оценок** по дисциплинам.  

---

## Используемые технологии
- Язык программирования: **Python 3**.  
- Библиотеки:
  - `socket` — для работы с сетевыми соединениями;  
  - `threading` — для реализации многопользовательского чата;  
  - `urllib.parse` — для обработки данных POST-запросов.  

---

## Структура проекта
- `task1` – UDP клиент/сервер.  
- `task2` – TCP клиент/сервер с вычислениями.  
- `task3` – HTTP-сервер, отдающий HTML-страницу.  
- `task4` – многопользовательский чат.  
- `task5` – веб-сервер для учёта оценок (GET/POST).  
- `lab1_report` – отчёт и документация (MkDocs).
