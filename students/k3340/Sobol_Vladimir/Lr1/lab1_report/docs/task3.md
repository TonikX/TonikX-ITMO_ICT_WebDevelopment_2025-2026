# Задание 3

## Условие
Реализовать серверное приложение.  
Клиент подключается к серверу и получает **HTTP-сообщение**, которое содержит HTML-страницу, подгружаемую сервером из файла `index.html`.

### Требования
- Использовать библиотеку **socket**.
- Реализовать через **TCP**.
- Сервер должен корректно формировать HTTP-заголовки и отдавать содержимое HTML-файла.

### Полезные ссылки
- [ZetCode: Работа с сокетами](https://zetcode.com/python/socket/)

---

## Решение
- Создан TCP-сервер, работающий на порту 8080.  
- При подключении клиент отправляет HTTP-запрос (например, через браузер).  
- Сервер считывает содержимое файла `index.html` и формирует корректный HTTP-ответ с заголовками.  
- В ответ клиенту отправляется HTML-страница, которая отображается в браузере.  
- Работа проверена: при открытии [http://localhost:8080](http://localhost:8080) загружается страница из `index.html`.

---

## Вывод
- Реализован простой HTTP-сервер средствами Python и библиотеки **socket**.  
- Закреплены знания структуры HTTP-запроса и ответа.  
- Получен базовый опыт отдачи статических файлов клиенту.
