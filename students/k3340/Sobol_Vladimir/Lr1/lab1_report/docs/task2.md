# Задание 2

## Условие
Реализовать клиентскую и серверную часть приложения.  
Клиент вводит параметры математической задачи, сервер обрабатывает данные и возвращает результат.  
Протокол обмена — **TCP**, библиотека — **socket**.

Варианты операций:
1. Теорема Пифагора  
2. Решение квадратного уравнения  
3. Площадь трапеции  
4. Площадь параллелограмма  

---

## Решение
- Выбран **вариант 1 — теорема Пифагора**.  
- Реализован TCP-сервер, который принимает от клиента два числа (катеты `a` и `b`).  
- Сервер вычисляет гипотенузу `c = sqrt(a² + b²)` и отправляет результат обратно клиенту.  
- Клиент подключается к серверу, вводит параметры и выводит ответ в консоль.  
- Проверка прошла успешно: введённые числа корректно обрабатываются, результат вычисляется и передаётся обратно.

---

## Вывод
- Получен опыт работы с **TCP-сокетами** в Python.  
- Реализован обмен данными по сети с вычислением результата на сервере.  
- Закреплены навыки построения клиент-серверного взаимодействия на основе **socket + TCP**.
