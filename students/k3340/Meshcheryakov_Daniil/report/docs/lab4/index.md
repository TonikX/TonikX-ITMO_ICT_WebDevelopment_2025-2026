# Лабораторная работа №4: Vue.js

**Клиентская часть системы управления читальным залом**

## Описание проекта

Реализация современного веб-интерфейса для системы управления читальным залом с использованием Vue.js 3, Vuetify 3 и интеграцией с REST API бэкенда.

## Цель работы

Реализация клиентской части приложения средствами Vue.js и настройка взаимодействия с серверной частью.

## Вариант задания

**Предметная область:** Читальный зал

**Интерфейсы:**
- Авторизация и регистрация пользователей
- Панель управления с общей статистикой
- Управление читальными залами (CRUD)
- Управление читателями (CRUD)
- Управление бронированиями (CRUD)
- Управление библиотекарями и расписанием (CRUD)
- Просмотр квартальных отчетов
- Поиск свободных залов

## Интеграция с Backend

### CORS Configuration
Backend настроен для приема запросов с фронтенда:

```python
# Django settings.py
CORS_ALLOWED_ORIGINS = [
    "http://localhost:3000",
]
```

### JWT Authentication Flow

1. **Вход:**
   - POST `/api/auth/jwt/create/`
   - Получение access и refresh токенов
   - Сохранение в Pinia store

2. **Запросы к API:**
   - Добавление `Authorization: Bearer <token>` header
   - Axios interceptor автоматически

3. **Обновление токена:**
   - При ошибке 401
   - POST `/api/auth/jwt/refresh/`
   - Повтор оригинального запроса

4. **Выход:**
   - Удаление токенов из store
   - Редирект на `/`
