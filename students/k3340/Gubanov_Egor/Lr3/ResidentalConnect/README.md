# ЖК Коннект

Информационная система для автоматизации работы Управляющей Компании ЖК.

## Описание

**ЖК Коннект** — это серверная часть (API) для SaaS-платформы, позволяющей автоматизировать взаимодействие между жильцами, диспетчерами и мастерами обслуживающей организации.

### Ключевой функционал

1. **Реестр недвижимости**: Учет жилых комплексов, домов и квартир с привязкой собственников.
2. **Тикет-система**: Подача заявок на ремонт жильцами, их модерация диспетчером и назначение мастерам.
3. **Учет ресурсов**: Передача и хранение истории показаний счетчиков (вода, электричество, газ).
4. **Ролевая модель**: Разделение прав доступа (Жилец/Мастер/Диспетчер).

## Технологии

- Django 5.2.8
- Django REST Framework 3.16.1
- Djoser 2.2.0
- django-filter 25.2
- MkDocs 1.5.3

## Установка

1. Установите зависимости:
```bash
pip install -r requirements.txt
```

2. Примените миграции:
```bash
python manage.py migrate
```

3. Создайте суперпользователя:
```bash
python manage.py createsuperuser
```

4. Запустите сервер:
```bash
python manage.py runserver
```

## API Endpoints

- `/api/buildings/` — управление домами
- `/api/apartments/` — управление квартирами
- `/api/categories/` — категории услуг
- `/api/service-requests/` — заявки на обслуживание
- `/api/meter-readings/` — показания счетчиков
- `/api/auth/` — аутентификация и управление пользователями

## Документация

Для просмотра документации используйте MkDocs:

```bash
mkdocs serve
```

Документация будет доступна по адресу: `http://127.0.0.1:8000`

## Роли пользователей

- **Жилец (resident)**: Просмотр своих данных, создание заявок, передача показаний
- **Мастер (master)**: Просмотр назначенных заявок, изменение статуса, добавление комментариев
- **Диспетчер (dispatcher)**: Полный доступ ко всем операциям
