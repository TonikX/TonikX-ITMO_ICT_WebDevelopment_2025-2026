    Устранение проблем с отображением диаграмм¶

    Проблема: Диаграммы Mermaid отображаются как код¶

    Если диаграммы Mermaid не отображаются визуально, а показываются только как код, попробуйте следующее:

    Решение 1: Обновить страницу с очисткой кеша¶

    • Windows/Linux: Ctrl + F5 или Ctrl + Shift + R
    • Mac: Cmd + Shift + R

    Решение 2: Проверить консоль браузера¶

    1. Откройте консоль разработчика (F12)
    2. Перейдите на вкладку "Console"
    3. Проверьте наличие ошибок JavaScript
    4. Если есть ошибки, сообщите разработчику

    Решение 3: Проверить подключение к интернету¶

    Mermaid диаграммы загружаются с CDN (unpkg.com). Убедитесь, что у вас есть доступ к интернету.

    Решение 4: Использовать текстовые схемы¶

    В разделе "Схемы процессов" есть текстовые версии всех схем в конце страницы. Они помогут понять процесс, даже если визуальные диаграммы не работают.

    Альтернативный способ просмотра¶

    Если диаграммы все еще не работают, вы можете:

    1. Скопировать код диаграммы Mermaid
    2. Открыть https://mermaid.live/
    3. Вставить код
    4. Просмотреть визуализацию

    Проверка работы MkDocs¶

    Убедитесь, что MkDocs запущен:

    cd /path/to/project
source venv/bin/activate
mkdocs serve

    Сервер должен быть доступен на http://127.0.0.1:8001/
