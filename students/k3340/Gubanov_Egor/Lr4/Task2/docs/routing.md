# Роутинг

## Обзор

Приложение использует Vue Router 4 для навигации между страницами.

## Конфигурация роутера

Роутер настроен в файле `src/router/index.js` с использованием:
- `createRouter` — создание экземпляра роутера
- `createWebHistory` — использование HTML5 History API

## Маршруты

### `/hi`

**Компонент:** `Hello.vue`

Простая страница с приветствием. Демонстрирует базовую работу роутинга.

### `/warriors`

**Компонент:** `Warriors.vue`

Главная страница приложения со списком воинов и формой создания.

## Навигация

Для программной навигации используйте:

```javascript
this.$router.push('/warriors')
```

Или в шаблоне:

```vue
<router-link to="/warriors">Воины</router-link>
```

## История навигации

Роутер использует `createWebHistory`, что означает:
- Чистые URL без хэша (`/warriors` вместо `/#/warriors`)
- Поддержка кнопок "Назад" и "Вперед" в браузере
- Возможность прямого доступа к URL

## Структура конфигурации

```javascript
const routes = [
  {
    path: '/hi',
    name: 'Hello',
    component: Hello
  },
  {
    path: '/warriors',
    name: 'Warriors',
    component: Warriors
  }
]
```

Каждый маршрут содержит:
- `path` — URL путь
- `name` — имя маршрута (опционально, для программной навигации)
- `component` — компонент Vue для отображения
