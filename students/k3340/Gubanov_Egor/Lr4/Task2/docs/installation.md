# Установка и запуск

## Требования

- Node.js версии 16.0 или выше
- npm или yarn

## Установка зависимостей

```bash
npm install
```

Это установит все необходимые зависимости, указанные в `package.json`:
- vue
- vue-router
- axios
- vite
- @vitejs/plugin-vue

## Запуск dev-сервера

```bash
npm run dev
```

После запуска приложение будет доступно по адресу:
- `http://localhost:5173`

Vite автоматически откроет браузер с приложением.

## Сборка для production

Для создания production-сборки выполните:

```bash
npm run build
```

Собранные файлы будут находиться в папке `dist/`.

## Просмотр production-сборки

Для локального просмотра production-сборки:

```bash
npm run preview
```

## Решение проблем

### Порт 5173 занят

Если порт 5173 уже занят, Vite автоматически предложит использовать другой порт. Или вы можете указать порт вручную в `vite.config.js`:

```javascript
server: {
  port: 3000
}
```

### Ошибки при установке зависимостей

Убедитесь, что у вас установлена актуальная версия Node.js:

```bash
node --version
```

Рекомендуется использовать Node.js версии 16.0 или выше.

### Проблемы с CORS при работе с API

Если возникают проблемы с CORS при обращении к API, убедитесь, что сервер `http://62.109.28.95:8890` доступен и разрешает запросы с вашего домена.
