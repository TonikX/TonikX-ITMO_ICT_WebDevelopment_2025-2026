# Vue Warriors App

Приложение на Vue.js 3 для работы со списком воинов, созданное в рамках практической работы 4.2 "Введение во Vue.js".

## Обзор проекта

**Vue Warriors App** — это простое одностраничное приложение (SPA), демонстрирующее базовые возможности Vue.js 3:
- Роутинг с помощью Vue Router
- Компонентная архитектура
- Работа с внешним API через axios
- Использование Options API

## Ключевой функционал

1. **Роутинг**: Навигация между страницами `/hi` и `/warriors`
2. **Список воинов**: Получение и отображение списка воинов из API
3. **Создание воина**: Форма для добавления нового воина в систему

## Технологии

- **Vue.js 3.4.0** — прогрессивный JavaScript-фреймворк
- **Vue Router 4.2.5** — официальный роутер для Vue.js
- **Axios 1.6.0** — HTTP-клиент для работы с API
- **Vite 5.0.0** — инструмент сборки и dev-сервер

## Структура приложения

Приложение состоит из следующих компонентов:

- **Hello.vue** — простой компонент приветствия
- **Warriors.vue** — главная страница со списком воинов
- **WarriorList.vue** — компонент для отображения списка
- **WarriorForm.vue** — форма создания нового воина

## Быстрый старт

1. Установите зависимости: `npm install`
2. Запустите dev-сервер: `npm run dev`
3. Откройте браузер: `http://localhost:5173`

Подробнее см. раздел [Установка](installation.md).

## Роуты

- `/hi` — страница с компонентом Hello
- `/warriors` — страница со списком воинов и формой создания

## API

Приложение использует внешний API:
- `GET http://62.109.28.95:8890/warrior/list` — получение списка воинов
- `POST http://62.109.28.95:8890/warrior/create1` — создание нового воина
