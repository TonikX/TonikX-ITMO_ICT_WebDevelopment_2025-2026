# Vue Warriors App

Приложение на Vue.js 3 для работы со списком воинов. Проект создан в рамках практической работы 4.2 "Введение во Vue.js".

## Описание

Приложение демонстрирует базовые возможности Vue.js 3:
- Роутинг с помощью Vue Router
- Компонентная архитектура
- Работа с API через axios
- Options API

## Установка

1. Установите зависимости:
```bash
npm install
```

2. Запустите dev-сервер:
```bash
npm run dev
```

Приложение будет доступно по адресу `http://localhost:5173`

## Роутинг

- `/hi` - страница с компонентом Hello
- `/warriors` - страница со списком воинов

## Компоненты

### Hello.vue
Простой компонент, отображающий приветствие.

### Warriors.vue
Главная страница приложения, содержащая:
- Кнопку для получения списка воинов
- Форму создания нового воина (WarriorForm)
- Список воинов (WarriorList)

### WarriorList.vue
Компонент для отображения списка воинов. Принимает props:
- `warriors` (Array) - массив воинов

Отображает для каждого воина:
- Имя (name)
- Расу (race)

### WarriorForm.vue
Форма для создания нового воина. Содержит два поля:
- Имя (name)
- Раса (race)

Отправляет POST запрос на `http://62.109.28.95:8890/warrior/create1`

## API

Приложение использует внешний API:
- GET `http://62.109.28.95:8890/warrior/list` - получение списка воинов
- POST `http://62.109.28.95:8890/warrior/create1` - создание нового воина

## Технологии

- Vue.js 3.4.0
- Vue Router 4.2.5
- Axios 1.6.0
- Vite 5.0.0
