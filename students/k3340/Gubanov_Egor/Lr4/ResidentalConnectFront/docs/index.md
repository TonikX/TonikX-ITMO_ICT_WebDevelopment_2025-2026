# ЖК Коннект - Frontend

Добро пожаловать в документацию фронтенд приложения для системы управления жилым комплексом "ЖК Коннект".

## Описание

Это веб-приложение разработано на **Vue.js 3** с использованием **Vuetify** для создания современного и удобного интерфейса. Приложение обеспечивает взаимодействие пользователей (жильцов, мастеров и диспетчеров) с бэкенд API для управления жилым комплексом.

## Основные возможности

- **Аутентификация и авторизация**
  - Вход в систему
  - Регистрация новых пользователей
  - Управление профилем
  - Смена пароля

- **Управление заявками**
  - Создание заявок на обслуживание
  - Просмотр и управление заявками
  - Назначение мастеров (для диспетчера)
  - Изменение статусов (для мастера/диспетчера)
  - Статистика по заявкам

- **Показания счетчиков**
  - Подача показаний счетчиков
  - Просмотр истории показаний
  - Статистика потребления

- **Управление квартирами**
  - Просмотр квартир
  - Создание и редактирование (для диспетчера)
  - Просмотр детальной информации

- **Управление домами** (только для диспетчера)
  - Создание и управление домами
  - Статистика по домам и квартирам

- **Современный UI**
  - Material Design компоненты (Vuetify)
  - Адаптивный дизайн
  - Интуитивный интерфейс
  - Dashboard с виджетами

- **Интеграция с API**
  - Работа с Django REST Framework
  - Token-based аутентификация
  - Обработка ошибок
  - Фильтрация и поиск

## Технологический стек

- **Vue.js 3** - прогрессивный JavaScript фреймворк
- **Vuetify 3** - Material Design компоненты
- **Vue Router 4** - роутинг
- **Pinia** - управление состоянием
- **Axios** - HTTP клиент
- **Vite** - инструмент сборки

## Быстрый старт

1. Установите зависимости:
```bash
npm install
```

2. Убедитесь, что бэкенд запущен на `http://127.0.0.1:8000`

3. Запустите сервер разработки:
```bash
npm run dev
```

4. Откройте браузер по адресу `http://localhost:5173`
