# Установка

## Предварительные требования

Перед установкой убедитесь, что у вас установлены:

- **Node.js** версии 16 или выше
- **npm** (обычно устанавливается вместе с Node.js)

Проверить версии можно командами:
```bash
node --version
npm --version
```

## Установка зависимостей

1. Перейдите в директорию проекта:
```bash
cd ResidentalConnectFront
```

2. Установите все зависимости:
```bash
npm install
```

Это установит все необходимые пакеты, указанные в `package.json`:
- Vue.js 3
- Vuetify 3
- Vue Router 4
- Pinia
- Axios
- Vite и плагины

## Настройка окружения

### Настройка API URL

По умолчанию приложение использует `http://127.0.0.1:8000` как базовый URL для API.

Для изменения создайте файл `.env` в корне проекта:

```env
VITE_API_BASE_URL=http://127.0.0.1:8000
```

Или для production:
```env
VITE_API_BASE_URL=https://your-api-domain.com
```

## Запуск проекта

### Режим разработки

Запустите сервер разработки:
```bash
npm run dev
```

Приложение будет доступно по адресу `http://localhost:5173`

Vite автоматически откроет браузер. Если этого не произошло, откройте адрес вручную.

### Режим production

Для сборки production версии:
```bash
npm run build
```

Собранные файлы будут в папке `dist/`. Эти файлы можно развернуть на любом веб-сервере.

Для предпросмотра production сборки:
```bash
npm run preview
```

## Проверка установки

После установки и запуска:

1. Откройте `http://localhost:5173` в браузере
2. Вы должны увидеть страницу входа
3. Убедитесь, что бэкенд сервер запущен на `http://127.0.0.1:8000`

## Решение проблем

### Ошибка порта занят

Если порт 5173 занят, Vite автоматически предложит использовать другой порт. Или измените порт в `vite.config.js`:

```javascript
server: {
  port: 3000, // другой порт
}
```

### Ошибки при установке зависимостей

Попробуйте:
```bash
rm -rf node_modules package-lock.json
npm install
```

### Проблемы с CORS

Убедитесь, что в Django бэкенде настроен CORS:
- `django-cors-headers` установлен
- `CORS_ALLOW_ALL_ORIGINS = True` в `settings.py` (для разработки)
