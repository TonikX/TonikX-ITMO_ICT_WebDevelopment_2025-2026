    Функциональность приложения

    Обзор

    Веб-приложение "ЖК Коннект" предоставляет полный набор интерфейсов для управления жилым комплексом с разделением прав доступа по ролям пользователей.

    Роли и права доступа

    Жилец (Resident)

    Доступные функции: - Просмотр своих квартир - Создание заявок на обслуживание для своих квартир - Просмотр своих заявок - Подача показаний счетчиков для своих квартир - Просмотр истории показаний - Редактирование профиля

    Ограничения: - Не может видеть чужие квартиры - Не может видеть чужие заявки - Не может изменять статус заявок - Не может назначать мастеров

    Мастер (Master)

    Доступные функции: - Просмотр заявок, назначенных ему - Изменение статуса назначенных заявок - Добавление комментариев к назначенным заявкам - Просмотр категорий услуг - Редактирование профиля

    Ограничения: - Не может видеть все заявки - Не может назначать других мастеров - Не может создавать заявки - Не может управлять квартирами и домами

    Диспетчер (Dispatcher)

    Доступные функции: - Полный доступ ко всем ресурсам - Создание и управление домами - Создание и управление квартирами - Просмотр всех заявок - Назначение мастеров на заявки - Изменение статусов заявок - Просмотр всех показаний счетчиков - Редактирование показаний - Просмотр статистики - Редактирование профиля

    Основные модули

    1. Аутентификация

    • Вход (/login) - авторизация пользователя
    • Регистрация (/register) - создание нового аккаунта
    • Профиль (/profile) - просмотр и редактирование данных пользователя
    • Смена пароля - изменение пароля в профиле

    2. Заявки на обслуживание

    Для жильца: - Создание заявок (/service-requests/new) - Просмотр своих заявок (/service-requests/my) - Просмотр деталей заявки

    Для мастера: - Просмотр назначенных заявок (/service-requests/assigned) - Изменение статуса заявок - Добавление комментариев

    Для диспетчера: - Просмотр всех заявок (/service-requests) - Назначение мастеров - Изменение статусов - Статистика по заявкам (/service-requests/statistics)

    Функции: - Фильтрация по статусу, категории, квартире - Поиск по тексту - Сортировка - Пагинация

    3. Показания счетчиков

    Для жильца: - Подача показаний (/meter-readings/new) - Просмотр своих показаний (/meter-readings) - Просмотр истории

    Для диспетчера: - Просмотр всех показаний - Редактирование показаний - Удаление показаний - Статистика (/meter-readings/statistics)

    Типы счетчиков: - Горячая вода - Холодная вода - Электричество - Газ

    4. Квартиры

    Для жильца: - Просмотр своих квартир (/apartments) - Просмотр деталей квартиры

    Для диспетчера: - Просмотр всех квартир - Создание квартир (/apartments/new) - Редактирование квартир - Просмотр деталей

    Информация о квартире: - Номер квартиры - Дом - Этаж - Площадь - Количество комнат - Баланс счета - Владелец

    5. Дома (только для диспетчера)

    • Просмотр списка домов (/buildings)
    • Создание домов (/buildings/new)
    • Редактирование домов
    • Просмотр деталей
    • Статистика (/buildings/statistics)

    Информация о доме: - Адрес - Количество этажей - Описание - Список квартир

    6. Dashboard

    Главная страница с виджетами в зависимости от роли:

    Для жильца: - Количество моих заявок - Количество показаний счетчиков - Количество моих квартир - Быстрые действия (подать заявку, подать показания) - Последние заявки

    Для мастера: - Количество назначенных заявок - Количество всех заявок - Список назначенных заявок

    Для диспетчера: - Статистика по домам, квартирам, заявкам, показаниям - Быстрые действия (создать дом, квартиру, статистика) - Последние заявки - Краткая статистика по статусам заявок

    Фильтрация и поиск

    Все списки поддерживают: - Фильтрацию по различным полям (статус, категория, дата и т.д.) - Поиск по текстовым полям - Сортировку по различным полям - Пагинацию для больших списков

    Обработка ошибок

    • Централизованная обработка ошибок API
    • Отображение ошибок валидации в формах
    • Уведомления об успешных операциях
    • Подтверждения для удаления

    Адаптивность

    Приложение адаптируется под различные размеры экранов: - Desktop (полная функциональность) - Tablet (оптимизированный интерфейс) - Mobile (компактный интерфейс)
