    Компоненты

    Обзор

    Приложение использует компонентный подход Vue.js. Основные компоненты разделены на Views (страницы) и переиспользуемые Components.

    Views (Страницы)

    Аутентификация

    Login.vue

    Страница входа в систему.

    Функциональность: - Форма входа (username, password) - Валидация полей - Обработка ошибок - Редирект после успешного входа

    Маршрут: /login

    Требования: Доступна только для неавторизованных пользователей

    Register.vue

    Страница регистрации нового пользователя.

    Функциональность: - Форма регистрации со всеми полями пользователя - Выбор роли (жилец, мастер, диспетчер) - Валидация данных - Автоматический вход после регистрации

    Поля формы: - Username (обязательно) - Email (обязательно) - Password (обязательно) - First name - Last name - Role (обязательно) - Phone number - Address

    Маршрут: /register

    Profile.vue

    Страница профиля пользователя с возможностью редактирования.

    Функциональность: - Просмотр данных пользователя - Редактирование профиля - Смена пароля - Валидация и обработка ошибок

    Вкладки: 1. Профиль - редактирование данных пользователя 2. Смена пароля - изменение пароля

    Маршрут: /profile

    Service Requests (Заявки)

    ServiceRequestsList.vue

    Список всех заявок с фильтрацией и поиском.

    Функциональность: - Отображение списка заявок - Фильтрация по статусу и категории - Поиск по тексту - Сортировка и пагинация - Переход к детальной информации

    Маршрут: /service-requests

    ServiceRequestForm.vue

    Форма создания/редактирования заявки.

    Функциональность: - Создание новой заявки - Редактирование существующей заявки - Выбор категории и квартиры - Установка приоритета

    Маршруты: /service-requests/new, /service-requests/:id/edit

    ServiceRequestDetail.vue

    Детальная информация о заявке с возможностью управления.

    Функциональность: - Просмотр полной информации о заявке - Назначение мастера (для диспетчера) - Изменение статуса (для мастера/диспетчера) - Добавление комментария (для мастера/диспетчера) - Редактирование и удаление

    Маршрут: /service-requests/:id

    MyRequests.vue

    Страница "Мои заявки" для жильца.

    Функциональность: - Отображение всех заявок текущего пользователя - Быстрое создание новой заявки

    Маршрут: /service-requests/my

    AssignedRequests.vue

    Страница "Назначенные мне" для мастера.

    Функциональность: - Отображение заявок, назначенных текущему мастеру

    Маршрут: /service-requests/assigned

    ServiceRequestsStatistics.vue

    Статистика по заявкам.

    Функциональность: - Общая статистика (всего, выполнено, в работе, новых) - Распределение по статусам - Распределение по приоритетам - Распределение по категориям - Среднее время решения

    Маршрут: /service-requests/statistics

    Meter Readings (Показания счетчиков)

    MeterReadingsList.vue

    Список показаний счетчиков с фильтрацией.

    Функциональность: - Отображение списка показаний - Фильтрация по типу счетчика, квартире, дате - Поиск и сортировка - Пагинация

    Маршрут: /meter-readings

    MeterReadingForm.vue

    Форма подачи показаний счетчика.

    Функциональность: - Выбор квартиры и типа счетчика - Ввод значения - Указание даты подачи

    Маршруты: /meter-readings/new, /meter-readings/:id/edit

    MeterReadingDetail.vue

    Детальная информация о показаниях.

    Функциональность: - Просмотр информации о показаниях - Редактирование (для диспетчера) - Удаление (для диспетчера)

    Маршрут: /meter-readings/:id

    MeterReadingsStatistics.vue

    Статистика по показаниям счетчиков.

    Функциональность: - Общая статистика (всего показаний, общий расход) - Расход по типам счетчиков - Средние значения по типам - Топ квартир по расходу

    Маршрут: /meter-readings/statistics

    Apartments (Квартиры)

    ApartmentsList.vue

    Список квартир с фильтрацией.

    Функциональность: - Отображение списка квартир - Фильтрация по дому и этажу - Поиск - Для жильцов - только свои квартиры - Для диспетчера - все квартиры

    Маршрут: /apartments

    ApartmentForm.vue

    Форма создания/редактирования квартиры.

    Функциональность: - Создание новой квартиры - Редактирование существующей квартиры - Выбор дома - Указание характеристик квартиры

    Маршруты: /apartments/new, /apartments/:id/edit

    Требования: Только для диспетчера

    ApartmentDetail.vue

    Детальная информация о квартире.

    Функциональность: - Просмотр информации о квартире - Ссылки на заявки и показания по квартире - Редактирование (для диспетчера)

    Маршрут: /apartments/:id

    Buildings (Дома)

    BuildingsList.vue

    Список домов с возможностью управления.

    Функциональность: - Отображение списка домов - Поиск - Создание, редактирование, удаление домов

    Маршрут: /buildings

    Требования: Только для диспетчера

    BuildingForm.vue

    Форма создания/редактирования дома.

    Функциональность: - Создание нового дома - Редактирование существующего дома - Указание адреса, количества этажей, описания

    Маршруты: /buildings/new, /buildings/:id/edit

    Требования: Только для диспетчера

    BuildingDetail.vue

    Детальная информация о доме.

    Функциональность: - Просмотр информации о доме - Ссылка на список квартир в доме - Редактирование

    Маршрут: /buildings/:id

    Требования: Только для диспетчера

    BuildingsStatistics.vue

    Статистика по домам и квартирам.

    Функциональность: - Общая статистика (всего домов, квартир, заселенность) - Статистика по каждому дому - Распределение по этажам - Распределение по количеству комнат

    Маршрут: /buildings/statistics

    Требования: Только для диспетчера

    Home.vue

    Главная страница (Dashboard) с виджетами.

    Функциональность: - Виджеты со статистикой в зависимости от роли - Быстрые действия - Последние заявки/показания - Краткая статистика (для диспетчера)

    Маршрут: /

    Переиспользуемые компоненты

    CategorySelect.vue

    Компонент выбора категории услуги.

    Использование: 

    <CategorySelect
  v-model="categoryId"
  label="Категория услуги *"
  required
  :error-messages="errors.category"
/>

    Props: - modelValue - выбранная категория (ID) - label - метка поля - required - обязательное поле - disabled - отключено - errorMessages - сообщения об ошибках

    ApartmentSelect.vue

    Компонент выбора квартиры.

    Использование: 

    <ApartmentSelect
  v-model="apartmentId"
  label="Квартира *"
  :building-id="buildingId"
  required
/>

    Props: - modelValue - выбранная квартира (ID) - label - метка поля - buildingId - фильтр по дому (опционально) - required - обязательное поле - disabled - отключено - errorMessages - сообщения об ошибках

    BuildingSelect.vue

    Компонент выбора дома.

    Использование: 

    <BuildingSelect
  v-model="buildingId"
  label="Дом *"
  required
/>

    Props: - modelValue - выбранный дом (ID) - label - метка поля - required - обязательное поле - disabled - отключено - errorMessages - сообщения об ошибках

    WorkerSelect.vue

    Компонент выбора мастера.

    Использование: 

    <WorkerSelect
  v-model="workerId"
  label="Мастер"
/>

    Props: - modelValue - выбранный мастер (ID) - label - метка поля - required - обязательное поле - disabled - отключено - errorMessages - сообщения об ошибках

    StatusChip.vue

    Компонент отображения статуса заявки с цветом.

    Использование: 

    <StatusChip :status="request.status" />

    Props: - status - статус заявки (new, in_progress, done, canceled) - size - размер чипа (small, default, large)

    PriorityChip.vue

    Компонент отображения приоритета заявки с цветом.

    Использование: 

    <PriorityChip :priority="request.priority" />

    Props: - priority - приоритет (low, medium, high, urgent) - size - размер чипа

    StatisticsCard.vue

    Карточка со статистикой для dashboard.

    Использование: 

    <StatisticsCard
  title="Всего заявок"
  :value="100"
  icon="mdi-clipboard-list"
  color="primary"
/>

    Props: - title - заголовок карточки - value - значение (число или строка) - subtitle - подзаголовок (опционально) - icon - иконка Material Design - color - цвет карточки

    DataTable.vue

    Универсальная таблица с пагинацией и сортировкой.

    Использование: 

    <DataTable
  :headers="headers"
  :items="items"
  :loading="loading"
  :total-items="totalCount"
  @update:options="handleOptionsUpdate"
/>

    Props: - headers - массив заголовков колонок - items - массив данных - loading - состояние загрузки - title - заголовок таблицы - itemsPerPage - элементов на странице - totalItems - общее количество элементов

    Vuetify компоненты

    Приложение активно использует компоненты из библиотеки Vuetify:

    Формы

    • v-form - контейнер формы
    • v-text-field - текстовое поле
    • v-textarea - многострочное текстовое поле
    • v-select - выпадающий список
    • v-btn - кнопка

    Навигация

    • v-app-bar - верхняя панель навигации
    • v-navigation-drawer - боковое меню
    • v-menu - выпадающее меню
    • v-tabs - вкладки

    Обратная связь

    • v-alert - уведомления (ошибки, успех)
    • v-card - карточка для группировки контента
    • v-chip - чип для отображения статусов

    Layout

    • v-container - контейнер с отступами
    • v-row / v-col - сетка для размещения элементов
    • v-spacer - заполнитель пространства

    Таблицы

    • v-data-table - таблица с пагинацией и сортировкой

    Стилизация

    Компоненты используют: - Scoped styles - стили ограничены компонентом - Vuetify темы - Material Design цвета и стили - Иконки Material Design Icons - через @mdi/font

    Создание нового компонента

    1. Создайте файл в src/components/MyComponent.vue
    2. Используйте структуру: 
      <template>
  <!-- HTML разметка -->
</template>

<script>
export default {
  name: 'MyComponent',
  props: {
    // свойства компонента
  },
  data() {
    return {
      // локальное состояние
    }
  },
  methods: {
    // методы компонента
  }
}
</script>

<style scoped>
/* Стили компонента */
</style>
    3. Импортируйте и используйте в нужном view: 
      import MyComponent from '@/components/MyComponent.vue'
