# Лабораторная работа №4 — клиент на Vue 3 + Vuetify

Клиентская часть для бэкенда ЛР3 (Django REST + Djoser). Реализованы интерфейсы авторизации, профиля и CRUD-страницы основных сущностей школы: предметы, кабинеты, классы, учителя, ученики, расписание, оценки.

## Что есть
- Авторизация/регистрация (Djoser token auth), профиль текущего пользователя.
- Справочники: предметы, кабинеты, классы, учителя, ученики.
- Учебный процесс: расписание уроков, четвертные оценки.
- Валидация и отображение ошибок (в т.ч. серверных HTML-ошибок) во всех формах.

## Быстрый старт
- Бэкенд: `python manage.py runserver 8002` (из ЛР3, активировать `.venv`).
- Фронтенд: `npm run dev -- --host --port 5173` (папка `lr_4/frontend`).
- Открыть: `http://127.0.0.1:5173/`.
- Админка: `http://127.0.0.1:8002/admin/` (логин `donperenon`, пароль `Admin123!`, при необходимости сменить).

Подробнее о запуске и зависимостях — в разделе «Запуск».
