## Проекты

### 1. django_project_ivanov (Практические задания 2.1-2.2)

**Тема:** Управление данными об автовладельцах

**Модели:**
- User (расширенная модель AbstractUser)
- Avtomobil (автомобили)
- Vladenie (владение автомобилями)
- Voditelskoe_udostoverenie (водительские удостоверения)

**Что реализовано:**
- Функциональные представления для владельцев
- Классовые представления для автомобилей
- CRUD операции
- Формы регистрации и редактирования
- Админ-панель

**Запуск:**
```bash
cd django_project_ivanov
python manage.py runserver
# http://127.0.0.1:8000/
```

### 2. main_lab (Основная лабораторная работа)

**Тема:** Табло отображения информации об авиаперелетах

**Модели:**
- User (расширенная модель AbstractUser)
- Flight (рейсы)
- Reservation (бронирования)
- Review (отзывы)

**Что реализовано:**
- Регистрация пользователей с расширенными полями
- Просмотр и бронирование рейсов
- Система отзывов с рейтингами
- Поиск рейсов по параметрам
- Управление бронированиями
- Админ-панель для подтверждения броней

**Запуск:**
```bash
cd main_lab
source venv/bin/activate
python manage.py runserver
# http://127.0.0.1:8000/
```

**Тестовые данные:**

### 3. practicum_3_user_extension (Практическое задание №3)

**Тема:** Расширение модели пользователя

**Что добавлено:**
- Номер паспорта (passport_number)
- Домашний адрес (home_address)
- Национальность (nationality)

**Что реализовано:**
- Кастомная модель User на основе AbstractUser
- Настройка AUTH_USER_MODEL
- Обновление всех форм и шаблонов
- Кастомная админ-панель
- Миграция данных

**Запуск:**
```bash
cd practicum_3_user_extension/django_project_ivanov
python manage.py runserver
# http://127.0.0.1:8000/
```

## Документация

### Просмотр документации локально

```bash
cd lab_2

# Активировать виртуальное окружение для MkDocs
source venv_docs/bin/activate

# Запустить локальный сервер
mkdocs serve

# Открыть в браузере
# http://127.0.0.1:8000/
```

### Сборка документации

```bash
mkdocs build
```

### Публикация на GitHub Pages

```bash
mkdocs gh-deploy
```
