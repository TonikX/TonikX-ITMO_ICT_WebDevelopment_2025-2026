# о проекте

## цель лабораторной работы

данная лабораторная работа направлена на изучение веб-разработки с использованием Django framework. в процессе выполнения заданий студент получает практический опыт создания полнофункциональных веб-приложений с использованием современных технологий и паттернов разработки.

## изучаемые темы

**Django framework**
модели данных и ORM - работа с базами данных через объектно-реляционное отображение, представления - функциональные и классовые (Function-based и Class-based Views), шаблоны - Django Template Language для создания динамических страниц, формы - ModelForm для валидации и обработки данных, админ-панель - встроенная система администрирования, миграции - управление изменениями схемы базы данных.

**работа с моделями**
создание моделей данных - определение структуры данных, связи между моделями - ForeignKey, OneToOne, ManyToMany, расширение AbstractUser - кастомизация модели пользователя, валидация данных - проверка корректности данных на уровне модели, методы моделей - добавление бизнес-логики в модели.

**представления и формы**
Function-based Views - простые функции для обработки запросов, Class-based Views - переиспользуемые классы представлений (ListView, DetailView, CreateView, UpdateView, DeleteView), ModelForm - автоматическое создание форм из моделей, обработка форм - валидация, сохранение, обработка ошибок, контекст шаблонов - передача данных в шаблоны.

**аутентификация и авторизация**
система пользователей - регистрация, вход, выход, расширение User модели - добавление дополнительных полей, разграничение доступа - проверка прав пользователей, хеширование паролей - безопасное хранение паролей.

## технологический стек

Python 3.13 - основной язык программирования (все проекты), Django 5.2.7 - веб-framework (модели, представления, шаблоны), SQLite 3 - встроенная база данных (разработка и тестирование), Django ORM - объектно-реляционное отображение (запросы к БД), Django Forms - работа с формами (валидация, обработка), Django Admin - админ-панель (управление данными), Django Templates - шаблонизатор (динамические страницы), HTML5/CSS3 - фронтенд (интерфейс пользователя).

## образовательные цели

**теоретические знания**
понимание архитектуры MTV (Model-Template-View), знание принципов работы Django ORM, понимание паттернов Class-based Views, знание системы миграций Django, понимание принципов аутентификации и авторизации.

**практические навыки**
создание веб-приложений на Django, проектирование моделей данных, работа с формами и валидацией, настройка админ-панели, создание пользовательских интерфейсов, отладка и тестирование приложений.

## описание проектов

**проект 1: автовладельцы (практические задания 2.1-2.2)**
система управления данными об автовладельцах, автомобилях, владении и водительских удостоверениях. реализованы базовые CRUD операции, функциональные и классовые представления, формы для создания и редактирования данных, админ-панель с настройкой отображения. проект демонстрирует основы работы с Django: создание моделей, настройка связей между ними, работа с представлениями и шаблонами.

**проект 2: табло авиарейсов (основная лабораторная работа)**
полнофункциональное веб-приложение для управления информацией об авиарейсах. реализована система регистрации пользователей с расширенной моделью (AbstractUser), система бронирования мест на рейсах с проверкой доступности, система отзывов о рейсах с рейтингами, админ-панель для подтверждения бронирований, красивый адаптивный интерфейс. проект демонстрирует продвинутые возможности Django: расширение модели пользователя, сложные связи между моделями, аннотации и агрегация в запросах, работа с аутентификацией.

**проект 3: расширение модели пользователя (практическое задание №3)**
расширение проекта "автовладельцы" с добавлением кастомной модели пользователя. добавлены новые поля: номер паспорта, домашний адрес, национальность. полная интеграция с существующей системой, обновление всех представлений и шаблонов, настройка AUTH_USER_MODEL. проект демонстрирует миграцию существующего проекта на кастомную модель пользователя, что является важным навыком для реальных проектов.
