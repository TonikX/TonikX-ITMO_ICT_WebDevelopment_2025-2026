# лабораторная работа №2 по web-программированию

документация по выполненным заданиям лабораторной работы №2 по веб-программированию с использованием Django.

## обзор заданий

выполнено 3 комплексных задания, демонстрирующих различные аспекты веб-разработки на Django.

**практические задания 2.1-2.2: автовладельцы**
Django проект, модели данных, админ-панель, CRUD операции. управление данными об автовладельцах, автомобилях, владении и водительских удостоверениях. реализованы функциональные и классовые представления, формы ModelForm.

**основная лабораторная работа: табло авиарейсов**
Django веб-приложение, AbstractUser, система бронирования. полнофункциональное приложение для управления авиарейсами с возможностью регистрации пользователей, бронирования мест, написания отзывов. реализована расширенная модель пользователя с дополнительными полями.

**практическое задание №3: расширение модели пользователя**
AbstractUser, кастомная модель User, новые поля. расширение пользовательской модели в проекте "автовладельцы" с добавлением полей: номер паспорта, домашний адрес, национальность. полная интеграция с существующей системой.

## цели лабораторной работы

изучение Django framework, работа с моделями и ORM, создание веб-приложений, работа с формами и шаблонами, расширение модели пользователя, CRUD операции, многие-ко-многим связи.

## технологический стек

Python 3.13, Django 5.2.7, SQLite 3, HTML5, CSS3, Django ORM, Django Forms, Django Admin, Django Templates, AbstractUser, ModelForm, Function-based Views, Class-based Views.

## быстрый старт

для запуска любого из проектов активируйте виртуальное окружение, перейдите в соответствующую папку проекта, выполните `python manage.py runserver`, откройте браузер по адресу http://127.0.0.1:8000/.

## ключевые особенности

**модели и ORM**
создание сложных моделей данных, связи ForeignKey и ManyToMany, миграции базы данных, расширение AbstractUser.

**представления**
функциональные представления (Function-based Views), классовые представления (Class-based Views), ListView, DetailView, CreateView, UpdateView, DeleteView.

**формы**
ModelForm для работы с моделями, валидация данных, обработка POST/GET запросов, кастомные формы регистрации.

**админ-панель**
настройка Django Admin, кастомные отображения, inline редактирование, фильтры и поиск.
