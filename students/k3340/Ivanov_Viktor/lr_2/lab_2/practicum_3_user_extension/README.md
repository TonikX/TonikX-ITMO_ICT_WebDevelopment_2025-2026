# Лабораторная работа 2 - Django проект "Автовладельцы"

## Описание проекта

Проект Django для управления данными об автовладельцах, автомобилях, владении и водительских удостоверениях.

## Структура проекта

```
lab_2/
├── tutorial-env/              # Виртуальное окружение Python
├── django_project_ivanov/     # Основной Django проект
│   ├── db.sqlite3            # База данных SQLite
│   ├── manage.py             # Утилита управления Django
│   ├── templates/            # HTML-шаблоны
│   │   ├── owner_list.html  # Список владельцев
│   │   └── owner.html       # Детальная информация о владельце
│   ├── django_project_ivanov/  # Настройки проекта
│   │   ├── settings.py
│   │   └── urls.py
│   └── project_first_app/    # Приложение
│       ├── models.py         # Модели данных
│       ├── views.py          # Контроллеры (представления)
│       ├── admin.py          # Конфигурация админ-панели
│       └── urls.py           # URL-маршруты
└── README.md
```

## Модели данных

### 1. User (Пользователь-владелец) - Расширенная модель AbstractUser
**Стандартные поля Django:**
- `username` - Логин пользователя
- `first_name` - Имя
- `last_name` - Фамилия
- `email` - Email
- `password` - Пароль (хешированный)

**Дополнительные поля:**
- `patronymic` - Отчество (VARCHAR 30)
- `data_rozhdeniya` - Дата рождения (DATE)

**Новые поля (Практическое задание №3):**
- `passport_number` - Номер паспорта (VARCHAR 20) ⭐
- `home_address` - Домашний адрес (TEXT) ⭐
- `nationality` - Национальность (VARCHAR 50) ⭐

### 2. Avtomobil (Автомобиль)
- `id_avtomobilya` - ID автомобиля (первичный ключ)
- `gos_nomer` - Госномер (VARCHAR 15)
- `marka` - Марка (VARCHAR 20)
- `model` - Модель (VARCHAR 20)
- `cvet` - Цвет (VARCHAR 30)

### 3. Vladenie (Владение)
- `id_vladeltsa_avto` - ID записи владения (первичный ключ)
- `id_vladelca` - ID владельца (внешний ключ)
- `id_avtomobilya` - ID автомобиля (внешний ключ)
- `data_nachala` - Дата начала владения (DATETIME)
- `data_okonchaniya` - Дата окончания владения (DATETIME)

### 4. Voditelskoe_udostoverenie (Водительское удостоверение)
- `id_udostovereniya` - ID удостоверения (первичный ключ)
- `id_vladelca` - ID владельца (внешний ключ)
- `nomer_udostovereniya` - Номер удостоверения (VARCHAR 10)
- `tip` - Тип удостоверения (VARCHAR 10)
- `data_vydachi` - Дата выдачи (DATETIME)

## Доступные URL-адреса

### Пользователи-владельцы (функциональные представления):
- **Главная страница** - http://127.0.0.1:8000/
  - Список всех пользователей (владельцев автомобилей)
  - Отображает: ФИО, логин, email, дату рождения, национальность
  
- **Детальная информация о пользователе** - http://127.0.0.1:8000/owner/<id>/
  - Например: http://127.0.0.1:8000/owner/1/
  - Например: http://127.0.0.1:8000/owner/2/
  - Показывает все поля, включая **паспорт, адрес, национальность** ⭐

- **Регистрация пользователя** - http://127.0.0.1:8000/owner/create/
  - Форма регистрации с расширенными полями
  - Включает: логин, пароль, ФИО, email, дату рождения, **паспорт, адрес, национальность** ⭐
  - Автоматический вход после регистрации

### Автомобили (классовые представления):
- **Список автомобилей** - http://127.0.0.1:8000/avtomobil/
  - Все автомобили с действиями (просмотр, редактирование, удаление)

- **Детальная информация** - http://127.0.0.1:8000/avtomobil/<id>/
  - Например: http://127.0.0.1:8000/avtomobil/1/

- **Создание автомобиля** - http://127.0.0.1:8000/avtomobil/create/
  - Форма добавления нового автомобиля

- **Редактирование автомобиля** - http://127.0.0.1:8000/avtomobil/<id>/update/
  - Например: http://127.0.0.1:8000/avtomobil/1/update/

- **Удаление автомобиля** - http://127.0.0.1:8000/avtomobil/<id>/delete/
  - Например: http://127.0.0.1:8000/avtomobil/1/delete/
