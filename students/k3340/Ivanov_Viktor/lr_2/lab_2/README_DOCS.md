## Быстрый старт

### Просмотр документации локально

```bash
# 1. Перейдите в папку с документацией
cd lr_2/lab_2

# 2. Активируйте виртуальное окружение (если нужно)
source venv/bin/activate  # Mac/Linux
# или
venv\Scripts\activate     # Windows

# 3. Установите необходимые пакеты (если не установлены)
pip install mkdocs mkdocs-material

# 4. Запустите локальный сервер
mkdocs serve

# 5. Откройте в браузере
# http://127.0.0.1:8000/
```

### Сборка статического сайта

```bash
mkdocs build
```

Результат будет в папке `site/`

## Редактирование документации

### Как добавить/изменить страницу

1. Откройте нужный файл `.md` в папке `docs/`
2. Отредактируйте используя Markdown синтаксис
3. Сохраните файл
4. Если запущен `mkdocs serve`, изменения отобразятся автоматически

### Как добавить новый раздел

1. Создайте новый файл или папку в `docs/`
2. Добавьте ссылку в `mkdocs.yml` в секцию `nav:`
3. Проверьте результат командой `mkdocs serve`

### Полное руководство

Смотрите файл `MKDOCS_GUIDE.md` для детального руководства по работе с MkDocs.

## Публикация на GitHub Pages

### Автоматическая публикация

```bash
mkdocs gh-deploy
```

Эта команда:
1. Соберет статический сайт
2. Создаст/обновит ветку `gh-pages`
3. Опубликует на GitHub Pages

### Настройка GitHub Pages

1. Перейдите в Settings → Pages вашего репозитория
2. Source: Deploy from a branch
3. Branch: `gh-pages`
4. Folder: `/` (root)
5. Save

### URL документации

После публикации документация будет доступна по адресу:

```
https://DonPerenon.github.io/ITMO_ICT_WebDevelopment_2025-2026/
```

## Команды для работы

### Разработка

```bash
mkdocs serve              # Запуск dev-сервера с hot-reload
mkdocs serve -a 0.0.0.0:8000  # Доступ из сети
```

### Сборка

```bash
mkdocs build              # Сборка статического сайта
mkdocs build --clean      # Сборка с очисткой
mkdocs build --strict     # Сборка с ошибками при предупреждениях
```

### Публикация

```bash
mkdocs gh-deploy          # Публикация на GitHub Pages
mkdocs gh-deploy --force  # Принудительная публикация
```
