# Лабораторная работа 4 - Vue.js Frontend для системы управления авиакомпанией

## Описание

Клиентская часть приложения для управления авиакомпанией, реализованная на Vue.js 3 с использованием Vuetify 3.

## Реализованные интерфейсы

### Аутентификация
1. **Страница входа** (`/login`) - авторизация пользователей
2. **Страница регистрации** (`/register`) - регистрация новых пользователей
3. **Профиль пользователя** (`/profile`) - просмотр и редактирование данных пользователя

### Основные интерфейсы
1. **Главная панель** (`/`) - статистика по сотрудникам, самолетам в ремонте, популярным маркам
2. **Управление флотом** (`/fleet`) - статистика по маркам самолетов, список самолетов в ремонте
3. **Управление рейсами** (`/flights`) - список рейсов с фильтром по заполненности
4. **Детали рейса** (`/flights/:id`) - информация о доступных местах на рейсе
5. **Сотрудники** (`/employees`) - общее количество сотрудников

## Используемые технологии

- **Vue 3** - прогрессивный JavaScript фреймворк
- **Vue Router** - официальный роутер для Vue.js
- **Pinia** - хранилище состояния для Vue
- **Vuetify 3** - Material Design компоненты для Vue
- **Axios** - HTTP клиент для API запросов
- **Vite** - инструмент сборки

## Особенности реализации

1. **Защищенные маршруты** - все основные страницы требуют аутентификации
2. **Централизованное управление состоянием** - использование Pinia для хранения данных пользователя
3. **Interceptors** - автоматическое добавление токена аутентификации к запросам
4. **Responsive дизайн** - адаптивный интерфейс благодаря Vuetify
5. **Material Design** - современный и понятный UI
