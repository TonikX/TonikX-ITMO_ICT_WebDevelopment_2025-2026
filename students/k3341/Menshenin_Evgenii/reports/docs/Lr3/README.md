# Лабораторная работа 3: API Endpoints

## Описание API

Документация по API endpoints, используемым в лабораторной работе 3 для авиакомпании.

## Аутентификация

### Регистрация пользователя
**POST** `/auth/users/`

Регистрация нового пользователя в системе.

**Запрос:**
```json
{
  "username": "new_user",
  "password": "new_password",
  "email": "user@example.com",
  "first_name": "First",
  "last_name": "Last"
}
```

**Ответ:**
После успешной регистрации возвращается информация о созданном пользователе.

---

### Вход в систему
**POST** `/auth/token/login/`

Получение токена аутентификации для доступа к защищенным ресурсам.

**Запрос:**
```json
{
  "username": "mezhendosina",
  "password": "mezhendosina"
}
```

**Ответ:**
```json
{
  "auth_token": "c93685c440d80db11f2ad2dcc3bfab162d96c64f"
}
```

---

### Получение информации о текущем пользователе
**GET** `/auth/users/me/`

Получение информации о текущем аутентифицированном пользователе.

**Заголовки:**
```
Authorization: Token c93685c440d80db11f2ad2dcc3bfab162d96c64f
```

**Ответ:**
Информация о текущем пользователе, включая id, username, email, first_name и last_name.

## Функциональные endpoints

### Получение самой популярной марки самолета
**GET** `/api/mark/top/`

Возвращает самую популярную марку самолета (с наибольшим количеством экземпляров).

**Ответ:**
```json
{
  "top_mark": "Boeing 737"
}
```

---

### Получение всех марок самолетов
**GET** `/api/mark/all/`

Возвращает список всех марок самолетов с количеством самолетов каждой марки.

**Ответ:**
```json
{
  "marks": [
    {
      "mark": "Boeing 737",
      "count_boards": 5
    },
    {
      "mark": "Airbus A320",
      "count_boards": 3
    }
  ],
  "count_boards": 8
}
```

---

### Подбор маршрутов по заполненности
**POST** `/api/routes/pick/`

Возвращает список рейсов с коэффициентом заполнения ниже заданного значения.

**Запрос:**
```json
{
  "filled_less_than": 0.8
}
```

**Ответ:**
```json
{
  "flights": [
    {
      "id": 1,
      "plane": 1,
      "status": "scheduled",
      "departure_airport": "SVO",
      "destination_airport": "LED",
      "arrival_time": "2023-06-15T14:30:00Z",
      "arrival_time_fact": "2023-06-15T14:30:00Z",
      "departure_time": "2023-06-15T12:00:00Z",
      "departure_time_fact": "2023-06-15T12:00:00Z"
    }
  ]
}
```

---

### Получение доступных мест на рейсе
**GET** `/api/flights/{id}/available_seats/`

Возвращает список доступных мест для конкретного рейса.

**Параметры пути:**
- `id` - идентификатор рейса

**Ответ:**
```json
{
  "available_seats": [1, 5, 12, 15, 23]
}
```

---

### Получение самолетов в ремонте
**GET** `/api/planes/in_repair/`

Возвращает список самолетов, находящихся в ремонте.

**Ответ:**
```json
{
  "planes": [
    {
      "id": 1,
      "company": 1,
      "status": "repair",
      "flight_duration": 300,
      "mark": "Boeing 737",
      "last_technical_service": "2023-06-01T10:00:00Z"
    }
  ]
}
```
