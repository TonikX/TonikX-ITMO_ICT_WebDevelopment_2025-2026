# Форма регистрации пользователя

```
class RegisterForm(UserCreationForm):
    class Meta:
        model = User
        fields = ['username', 'email', 'password1', 'password2']
```

## Назначение

`RegisterForm` используется для создания нового пользователя в системе.
Эта форма отвечает за функцию «Регистрация новых пользователей», которая обязательна по заданию лабораторной работы.

## Основа

Класс наследуется от `UserCreationForm` (стандартная форма Django).
Это важно, потому что:

* уже реализована проверка на совпадение паролей (`password1` и `password2`);
* уже настроена валидация сложности пароля;
* при успешной отправке форма создаёт объект `User`.

То есть своя логика безопасности не пишется заново.

## Поля формы

* `username` — имя пользователя (логин, под которым он будет входить).
* `email` — контактная почта (может использоваться для связи или восстановления).
* `password1` — пароль.
* `password2` — повтор пароля. Django сам проверяет, что пароли совпадают.

## Что происходит при сохранении

После `form.is_valid()` вызов `form.save()` создаёт запись в таблице `auth_user`.
То есть человек становится полноценным пользователем системы и может:

* авторизоваться,
* регистрироваться на конференцию,
* оставлять отзывы.

## Вывод

`RegisterForm` закрывает пользовательскую часть аутентификации: она даёт интерфейс для самостоятельной регистрации без участия администратора, с минимально необходимыми полями и встроенной валидацией пароля.
