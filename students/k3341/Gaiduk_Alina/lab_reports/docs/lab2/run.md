# Инструкция по запуску проекта

## 1. Клонирование проекта

Склонировать репозиторий с кодом в любую локальную директорию.

```
git clone <ссылка-на-репозиторий>
cd scientific_conferences
```

Далее работа идёт из корневой папки проекта (`scientific_conferences/`, там где лежит `manage.py`).

---

## 2. Создание и активация виртуального окружения

Создать виртуальное окружение (venv):

```bash
python -m venv .venv
```

Активировать окружение:

**Windows (PowerShell):**

```powershell
.\.venv\Scripts\Activate.ps1
```
---

## 3. Установка зависимостей

Установить Django:
```
pip install django
```
---

## 4. Применение миграций БД

Создать / обновить структуру базы данных:

```bash
python manage.py migrate
```

---

## 5. Создание суперпользователя (админка)

Чтобы зайти в Django admin и выставлять результат выступления:

```bash
python manage.py createsuperuser
```

Далее ввести имя пользователя, почту и пароль.

---

## 6. Запуск сервера разработки

Запустить локальный сервер:

```bash
python manage.py runserver
```

После запуска проект доступен в браузере по адресу:
`http://127.0.0.1:8000/`
