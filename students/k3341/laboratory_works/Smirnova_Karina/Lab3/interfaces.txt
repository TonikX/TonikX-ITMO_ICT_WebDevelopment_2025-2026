1. GET /planes - посмотреть все самолеты
2. GET /planes/id - посмотреть самолет по id
3. POST /planes - создать самолет
4. PUT /planes/id - Отредактировать самолет по id
5. DELETE /planes/id - удалить самолет по id

8. POST /airline-companies - создать компанию
9. PUT /airline-companies/id - обновить компанию по id
10. DELETE /airline-companies/id - удалить компанию по id

11. GET /crew-members - получить всех сотрудников
12. GET /crew-members/id - получить сотрудника по id
13. POST /crew-members - создать сотрудника
14. PUT /crew-members/id - обновить сотрудника по id
