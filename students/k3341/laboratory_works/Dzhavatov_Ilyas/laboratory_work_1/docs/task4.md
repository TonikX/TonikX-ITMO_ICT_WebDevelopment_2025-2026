# Задание 4

## Условие
Реализовать двухпользовательский или многопользовательский чат. Для максимального количества баллов реализуйте многопользовательский чат.

### Требования
- Обязательно использовать библиотеку socket.
- Для многопользовательского чата необходимо использовать библиотеку threading.

---

## Решение
- Выбран протокол **TCP**.  
- **Сервер**:
  - Для каждого клиента создаётся поток:
    - принимает никнейм пользователя;
    - добавляет соединение в список клиентов;
    - читает сообщения и рассылает всем остальным;
    - удаляет клиента при выходе или обрыве соединения.
- **Клиент**:
  - Подключается к серверу, вводит никнейм.
  - Два потока:
    - `reader` — выводит все входящие сообщения;
    - `writer` — отправляет пользовательский ввод на сервер, команда `/quit` завершает работу.

---

## Проверка
1. Запустить сервер.  
2. Запустить несколько клиентов, ввести никнеймы.  
3. Сообщения одного клиента отображаются у всех остальных.  
4. При `/quit` клиент выходит, остальные получают уведомление.  

---

## Вывод
- Реализован многопользовательский чат на **TCP** с помощью **socket** и **threading**.  
- Поддерживается одновременное подключение нескольких пользователей и обмен сообщениями в реальном времени.
