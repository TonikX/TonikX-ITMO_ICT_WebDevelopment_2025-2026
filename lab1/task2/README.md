# Лабораторная работа 1, Задание 2: TCP клиент-сервер для математических операций

## Вариант 4: Поиск площади параллелограмма
**Позиция в журнале:** 12

## Описание
TCP приложение клиент-сервер для вычисления площади параллелограмма.
Клиент запрашивает ввод параметров с клавиатуры, сервер вычисляет результат.

### Формула
```
Площадь = основание × высота
```

## Файлы
- `server.py` - серверная часть (TCP сервер)
- `client.py` - клиентская часть (TCP клиент)

## Как запустить

### 1. Запустите сервер:
```bash
python3 server.py
```

### 2. В другом терминале запустите клиент:
```bash
python3 client.py
```

### 3. Введите параметры
Следуйте инструкциям клиента для ввода основания и высоты параллелограмма.

## Пример работы
```
=== Вычисление площади параллелограмма ===
Формула: Площадь = основание × высота

Введите основание параллелограмма: 5
Введите высоту параллелограмма: 3

Отправляем данные серверу: 5,3

Ответ сервера: Площадь параллелограмма: 15.0
```

## Технические детали
- **Протокол:** TCP (Transmission Control Protocol)
- **Порт сервера:** 12346
- **Адрес:** localhost (127.0.0.1)
- **Тип сокета:** `SOCK_STREAM`
- **Формат данных:** "основание,высота"

## Требования
- Python 3.x
- Библиотека `socket` (входит в стандартную библиотеку Python)

## Особенности TCP
- Надежное соединение
- Гарантированная доставка данных
- Установление соединения перед передачей данных
- Проверка ошибок и повторная передача при необходимости

## Реализация

### Обработка ошибок
```python
# Парсинг данных с обработкой ошибок
try:
    base, height = map(float, message.split(','))
    area = base * height
    result = f"Площадь параллелограмма: {area}"
except ValueError:
    result = "Ошибка: неверный формат данных. Используйте: основание,высота"
```

### Алгоритм работы

#### Сервер:
1. Создание TCP сокета (`SOCK_STREAM`)
2. Привязка к адресу и порту
3. Начало прослушивания (`listen()`)
4. Цикл принятия подключений (`accept()`)
5. Получение данных от клиента (`recv()`)
6. Парсинг данных в формате "основание,высота"
7. Вычисление площади: `площадь = основание × высота`
8. Отправка результата клиенту
9. Закрытие соединения

#### Клиент:
1. Создание TCP сокета
2. Подключение к серверу (`connect()`)
3. Запрос данных у пользователя
4. Формирование сообщения в формате "основание,высота"
5. Отправка данных серверу (`send()`)
6. Получение результата (`recv()`)
7. Отображение результата пользователю

### Валидация данных
- Проверка корректности ввода пользователя
- Обработка неверного формата данных
- Graceful обработка ошибок соединения

### Формат обмена данными
- **Отправка**: `"основание,высота"` (строка с запятой-разделителем)
- **Получение**: `"Площадь параллелограмма: {результат}"` или сообщение об ошибке
