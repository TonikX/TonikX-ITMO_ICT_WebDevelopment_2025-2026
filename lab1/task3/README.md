# Лабораторная работа 1, Задание 3: HTTP сервер

## Описание
Простой HTTP сервер, написанный на Python с использованием библиотеки `socket`.
Сервер принимает подключения клиентов и отвечает HTML-страницей, загруженной из файла `index.html`.

## Файлы
- `server.py` - HTTP сервер
- `index.html` - HTML-страница для отображения
- `README.md` - документация

## Как запустить

### 1. Запустите сервер:
```bash
python3 server.py
```

### 2. Откройте браузер и перейдите по адресу:
```
http://localhost:8080
```

### 3. Для остановки сервера:
Нажмите `Ctrl+C` в терминале

## Альтернативные способы тестирования

### Через curl:
```bash
curl http://localhost:8080
```

### Через wget:
```bash
wget -O - http://localhost:8080
```

## Технические детали

### Протокол и настройки
- **Протокол:** HTTP/1.1 over TCP
- **Порт:** 8080
- **Адрес:** localhost (127.0.0.1)
- **Тип сокета:** `SOCK_STREAM`
- **Кодировка:** UTF-8

### HTTP заголовки
Сервер отправляет следующие заголовки:
```http
HTTP/1.1 200 OK
Content-Type: text/html; charset=utf-8
Content-Length: [размер_файла]
Connection: close
```

### Обработка запросов
1. Сервер принимает TCP соединение
2. Читает HTTP запрос от клиента
3. Загружает содержимое `index.html`
4. Формирует корректный HTTP ответ
5. Отправляет ответ клиенту
6. Закрывает соединение

## Особенности реализации

### Загрузка HTML файла
```python
def load_html_file():
    try:
        with open('index.html', 'r', encoding='utf-8') as file:
            return file.read()
    except FileNotFoundError:
        return "<html><body><h1>Ошибка: index.html не найден</h1></body></html>"
```

### Формирование HTTP ответа
- Корректные HTTP заголовки
- Указание типа контента и кодировки
- Вычисление длины контента
- Разделение заголовков и тела ответа

### Обработка ошибок
- Обработка отсутствующего HTML файла
- Graceful shutdown по Ctrl+C
- Обработка исключений при работе с клиентами

## Требования
- Python 3.x
- Библиотека `socket` (входит в стандартную библиотеку Python)
- Файл `index.html` в той же папке что и `server.py`
