# Task 5: Журнал оценок - HTTP сервер на socket

## Описание

Простой веб-сервер для обработки GET и POST HTTP-запросов с помощью библиотеки socket в Python. Сервер реализует журнал оценок по дисциплинам с возможностью добавления новых оценок и просмотра всех существующих.

## Функциональность

Сервер поддерживает:

1. **GET запросы**: Отображение HTML-страницы с формой для добавления оценок и таблицей всех оценок
2. **POST запросы**: Добавление новых оценок по дисциплинам
3. **Структурированное хранение**: Оценки группируются по дисциплинам (несколько оценок на одну дисциплину)

## Особенности реализации

- **Структура данных**: Используется `defaultdict(list)` для хранения списка оценок по каждой дисциплине
- **HTTP парсинг**: Ручная обработка HTTP запросов через socket
- **HTML интерфейс**: Красивая веб-форма для ввода данных и таблица для отображения результатов
- **Валидация**: Проверка корректности введенных данных (оценки от 1 до 5)
- **Статистика**: Автоматический подсчет количества оценок и среднего балла по каждой дисциплине

## Структура данных

```python
grades_journal = {

```

## Запуск сервера

```bash
python3 server.py
```

Сервер запустится на `http://localhost:8080`

## Использование

1. **Открыть браузер** и перейти по адресу `http://localhost:8080`
2. **Добавить оценку**:
   - Введите название дисциплины
   - Введите оценку (от 1 до 5)
   - Нажмите "Добавить оценку"
3. **Просмотр результатов**: Все оценки отображаются в таблице с группировкой по дисциплинам

## Примеры запросов

### GET запрос (просмотр журнала)
```
GET / HTTP/1.1
Host: localhost:8080
```

### POST запрос (добавление оценки)
```
POST /add_grade HTTP/1.1
Host: localhost:8080
Content-Type: application/x-www-form-urlencoded
Content-Length: 31

discipline=Математика&grade=5
```

## Архитектура и реализация

### Структура HTTP сервера

```python
def parse_http_request(request_data):
    """Парсит HTTP запрос и возвращает метод, путь и тело запроса"""
    lines = request_data.split('\r\n')
    request_line = lines[0]
    method, path = request_line.split(' ')[:2]
    
    # Поиск тела запроса после пустой строки
    body = ""
    empty_line_found = False
    for line in lines:
        if empty_line_found:
            body += line + '\r\n'
        elif line == "":
            empty_line_found = True
    
    return method, path, body.strip()
```

### Обработка маршрутов

#### GET запросы (отображение страницы):
1. Генерация HTML-формы для ввода оценок
2. Создание таблицы всех существующих оценок
3. Подсчет статистики по каждой дисциплине
4. Формирование HTTP ответа с корректными заголовками

#### POST запросы (добавление оценок):
1. Парсинг HTTP запроса
2. Извлечение данных формы (`discipline` и `grade`)
3. Валидация оценки (1-5)
4. Добавление в структуру данных
5. Редирект на главную страницу

### HTTP заголовки

```python
def create_html_response(html_content):
    response_body = html_content.encode('utf-8')
    response_headers = f"""HTTP/1.1 200 OK\r
Content-Type: text/html; charset=utf-8\r
Content-Length: {len(response_body)}\r
Connection: close\r
\r
"""
    return response_headers.encode('utf-8') + response_body
```

### Валидация и безопасность

- **Валидация оценок**: Проверка диапазона 1-5
- **Обработка ошибок**: Graceful обработка неверных данных
- **Escape HTML**: Предотвращение XSS атак в пользовательском вводе
- **URL декодирование**: Корректная обработка символов в URL
