# MkDocs Документация - Руководство

## 🚀 Быстрый старт

### 1. Установка зависимостей

```bash
# Активируйте виртуальное окружение
venv\Scripts\activate

# Установите зависимости
pip install -r requirements.txt
```

### 2. Запуск документации

```bash
# Перейдите в папку проекта
cd car_owners_project

# Запустите MkDocs сервер
mkdocs serve
```

### 3. Открытие в браузере

Документация будет доступна по адресу: **http://127.0.0.1:8000**

## 🔧 Команды MkDocs

### Разработка

```bash
# Запуск сервера разработки
mkdocs serve

# Запуск с автоматическим обновлением
mkdocs serve --livereload

# Запуск на определенном порту
mkdocs serve --dev-addr=0.0.0.0:8080
```

### Сборка

```bash
# Сборка статических файлов
mkdocs build

# Сборка в определенную папку
mkdocs build --site-dir=../site
```

### Развертывание

```bash
# Развертывание на GitHub Pages
mkdocs gh-deploy

# Развертывание с принудительным обновлением
mkdocs gh-deploy --force
```

## 📝 Редактирование документации

### Добавление новой страницы

1. Создайте файл в папке `docs/`
2. Добавьте ссылку в `mkdocs.yml` в раздел `nav`
3. Перезапустите сервер

### Пример добавления страницы

```yaml
# mkdocs.yml
nav:
  - Главная: index.md
  - Новая страница: new-page.md  # Добавьте эту строку
```

### Создание файла

```markdown
# docs/new-page.md

# Новая страница

Содержимое новой страницы...
```
