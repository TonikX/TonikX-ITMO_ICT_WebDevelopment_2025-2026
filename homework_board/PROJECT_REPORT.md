# Отчет о выполнении проекта "Доска домашних заданий"

## 🎯 Цель проекта

Создание современной веб-системы для управления домашними заданиями в учебных заведениях с разделением ролей пользователей, поиском, пагинацией и современным дизайном.

## ✅ Выполненные требования

### 1. **Модель данных**
- ✅ **Предмет** - название, описание
- ✅ **Пользователь** - расширенная модель с ролями (студент/преподаватель/админ)
- ✅ **Задание** - предмет, преподаватель, дата выдачи, срок выполнения, описание, штрафы
- ✅ **Сдача** - связь студента с заданием, текст сдачи, дата, статус опоздания
- ✅ **Оценка** - баллы, комментарий преподавателя, дата оценки

### 2. **Функциональность**
- ✅ **Регистрация пользователей** с выбором роли
- ✅ **Просмотр заданий** по всем дисциплинам с сроками и описанием
- ✅ **Сдача заданий** в текстовом виде
- ✅ **Оценка заданий** преподавателями через админ-панель
- ✅ **Таблица оценок** всех студентов класса

### 3. **Дополнительные возможности**
- ✅ **Bootstrap меню** с адаптивной навигацией
- ✅ **Пагинация страниц** с сохранением параметров поиска
- ✅ **Поиск и фильтрация** по всем основным объектам
- ✅ **Современный дизайн** с CSS-анимациями и градиентами
- ✅ **Разделение прав** для разных ролей пользователей

## 🏗 Архитектура системы

### **Модели данных:**
```python
User (AbstractUser) - пользователи с ролями
├── Subject - предметы
├── Assignment - задания
│   └── Submission - сдачи заданий
│       └── Grade - оценки
```

### **Роли пользователей:**
- **Студент** - просмотр заданий, сдача работ, просмотр оценок
- **Преподаватель** - создание заданий, оценка работ, статистика
- **Администратор** - полный доступ ко всем функциям

### **URL-маршруты:**
- `/` - главная страница
- `/register/` - регистрация
- `/accounts/login/` - вход в систему
- `/assignments/` - список заданий
- `/assignments/<id>/` - детали задания
- `/assignments/<id>/submit/` - сдача задания
- `/grades/` - список оценок
- `/my-submissions/` - мои сдачи (для студентов)
- `/statistics/` - статистика (для преподавателей)
