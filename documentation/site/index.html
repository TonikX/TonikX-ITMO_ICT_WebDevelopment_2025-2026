Отчёт по лабораторным работам

по дисциплине «Web-программирование»

Автор: Ермаков Максим Олегович Группа: К3340

Введение

Данный отчёт содержит результаты выполнения лабораторных и практических работ по дисциплине «Web-программирование».
Цель курса — освоить принципы разработки клиент-серверных приложений на Python, включая работу с сетевыми протоколами, веб-серверами и современными веб-фреймворками.

В отчёт включены материалы двух лабораторных работ:

Лабораторная работа №1 — Клиент-серверные приложения на Python

Цель:
Освоить основы сетевого программирования с использованием библиотеки socket и реализовать простые сетевые приложения.

Основные задания: 1. UDP-клиент и сервер.
2. TCP-клиент и сервер для вычисления площади параллелограмма.
3. HTTP-сервер, возвращающий HTML-страницу.
4. Многопользовательский чат (TCP).
5. Мини-веб-сервер с обработкой GET и POST запросов.

Результат:
Разработаны и протестированы клиент-серверные приложения для различных протоколов, отработаны навыки работы с потоками и базовыми HTTP-заголовками.

Лабораторная работа №2 — Django Web Framework

Цель:
Изучить принципы построения веб-приложений с использованием Django-фреймворка.

Состав работы: 1. Практическая работа №1 — установка Django, создание проекта и моделей, настройка маршрутов и админ-панели.
2. Практическая работа №2 — реализация CRUD-операций (создание, чтение, редактирование, удаление) с применением FBV и CBV.
3. Практическая работа №3 — расширение пользовательской модели (AbstractUser), добавление полей и создание интерфейса регистрации пользователя.

Результат:
Создано полноценное Django-приложение с административной панелью, шаблонами и формами; реализованы CRUD-операции и кастомная модель пользователя.
